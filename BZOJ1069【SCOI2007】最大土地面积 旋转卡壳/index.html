BZOJ1069【SCOI2007】最大土地面积 <旋转卡壳>

Posted on 2018-02-09 Symbols count in article: 607 Reading time ≈ 2 mins.

Problem

【SCOI2007】最大土地面积

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

在某块平面土地上有NNN个点，你可以选择其中的任意四个点，使得这四个点围成的土地面积最大。

Input

第111行一个正整数NNN，接下来NNN行，每行222个数x,yx,yx,y，表示该点的横坐标和纵坐标。

Output

最大的多边形面积，答案精确到小数点后333位。

Sample Input

Sample Output

HINT

数据范围: n≤2000n\le 2000n≤2000，∣x∣,∣y∣≤105|x|,|y|\le 10^5∣x∣,∣y∣≤105

标签：旋转卡壳

Solution

基础旋转卡壳。

首先这四个点一定在凸包上，先求凸包。

考虑枚举每条对角线，找出其两边最远的点，即可找到该对角线对应的最大四边形。
这样就是一个O(n3)O(n^3)O(n3)的暴力。

枚举对角线的一段，移动另一端，发现两边最远的点都是单调移动的，于是可以带上旋转卡壳，这样内层循环的总复杂度时O(n)O(n)O(n)，就有了一个O(n2)O(n^2)O(n2)的优质算法。
