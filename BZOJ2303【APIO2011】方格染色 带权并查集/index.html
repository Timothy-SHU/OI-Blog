Memory  Limit:  256MB\mathrm{Memory\;Limit:\;256MB}MemoryLimit:256MB

Description

Sam\mathrm{Sam}Sam和他的妹妹Sara\mathrm{Sara}Sara有一个包含n×mn\times mn×m个方格的表格。她们想要将其的每个方格都染成红色或蓝色。
出于个人喜好，他们想要表格中每个2×22\times 22×2的方形区域都包含奇数个（111个或333个）红色方格。 可是昨天晚上，有人已经给表格中的一些方格染上了颜色！
现在Sam\mathrm{Sam}Sam和Sara\mathrm{Sara}Sara非常生气。不过，他们想要知道是否可能给剩下的方格染上颜色，使得整个表格仍然满足她们的要求。
如果可能的话，满足他们要求的染色方案数有多少呢？

Input

输入的第一行包含三个整数nnn, mmm和kkk，分别代表表格的行数、列数和已被染色的方格数目。
之后的kkk行描述已被染色的方格。其中第iii行包含三个整数xix_ixi​, yiy_iyi​和cic_ici​，分别代表第iii个已被染色的方格的行编号、列编号和颜色。cic_ici​为111表示方格被染成红色，cic_ici​为000表示方格被染成蓝色。

Output

输出一个整数，表示可能的染色方案数目WWW模10910^9109得到的值。

Solution

并查集解异或方程组。

于是我们尝试确定第一行和第一列的情况，即做一个n+m−1n+m-1n+m−1个变量的异或方程组。
