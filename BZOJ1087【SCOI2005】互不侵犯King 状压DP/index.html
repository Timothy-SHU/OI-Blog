Problem

【SCOI2005】互不侵犯King

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  162  MB\mathrm{Memory\;Limit:\;162\;MB}MemoryLimit:162MB

Description

在N×NN\times NN×N的棋盘里面放KKK个国王，使他们互不攻击，共有多少种摆放方案。国王能攻击到它上下左右，以及左上左下右上右下八个方向上附近的各一个格子，共888个格子。

Input

只有一行，包含两个数NNN, KKK (1≤N≤91 \le N \le 91≤N≤9, 0≤K≤N20 \le K \le N^20≤K≤N2)

Output

方案数。

Sample Input

Sample Output

标签：状压DP

Solution

状压DPDPDP的基础题。
对于每个国王，不难发现它放与不放只与当行和前一行有关，而它放完后的影响只作用于当行和后一行。
而N≤9N\le 9N≤9
于是考虑状压DPDPDP，f[i][j][k]f[i][j][k]f[i][j][k]表示当前考虑到第iii行，共用了jjj个国王，第iii行状态为kkk的方案数。那么kkk和前一行的状态k′k'k′是否冲突可以用kkk、k<<1k<<1k<<1、k>>1k>>1k>>1与k′k'k′取&\&&得到。时间复杂度O(n2×2k×2k)O(n^2\times 2^k\times 2^k)O(n2×2k×2k)。
