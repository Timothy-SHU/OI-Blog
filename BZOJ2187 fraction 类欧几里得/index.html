Description

给出444个正整数a,b,c,da,b,c,da,b,c,d，求一个最简分数pq\frac{p}{q}qp​满足ab<pq<cd\frac{a}{b}<\frac{p}{q}<\frac{c}{d}ba​<qp​<dc​。
若有多组解，输出qqq最小的一组，若仍有多组解，输出ppp最小的一组。

Input

Output

对于输入的每组数据输出一个最简分数pq\frac{p}{q}qp​。

Solution

类欧几里得基础题。

设f(a,b,c,d)f(a,b,c,d)f(a,b,c,d)为满足ab<pq<cd\frac{a}{b}<\frac{p}{q}<\frac{c}{d}ba​<qp​<dc​的p,qp,qp,q，考虑将其转化为规模更小的问题。

  • 若ab\frac{a}{b}ba​和cd\frac{c}{d}dc​间相差至少一个整数，则ppp为最小符合条件的整数，qqq为111
  • 若a≤b且c≤da\le b 且 c\le da≤b且c≤d，则一定有等价不等式ba>qp>dc\frac{b}{a}>\frac{q}{p}>\frac{d}{c}ab​>pq​>cd​，于是将问题转化为f(d,c,b,a)f(d,c,b,a)f(d,c,b,a)，求出qp\frac{q}{p}pq​后颠倒分子分母得到pq\frac{p}{q}qp​
  • 若a>ba>ba>b或c>dc>dc>d，令t=⌊ab⌋t=\lfloor\frac{a}{b}\rfloort=⌊ba​⌋，则有a−btb<p−qtq<c−dtd\frac{a-bt}{b}<\frac{p-qt}{q}<\frac{c-dt}{d}ba−bt​<qp−qt​<dc−dt​，问题转化为f(a−bt,b,c−dt,d)f(a-bt,b,c-dt,d)f(a−bt,b,c−dt,d)，求出p−qtq\frac{p-qt}{q}qp−qt​后即可还原出pq\frac{p}{q}qp​

由于每次均将参数变为其辗转相模的结果，故而复杂度和欧几里得算法大致相同。
