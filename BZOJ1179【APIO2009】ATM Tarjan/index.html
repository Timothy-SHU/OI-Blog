Problem

【APIO2009】ATM

Time  Limit:  15  Sec\mathrm{Time\;Limit:\;15\;Sec}TimeLimit:15Sec
Memory  Limit:  162  MB\mathrm{Memory\;Limit:\;162\;MB}MemoryLimit:162MB

Description

Siruseri\mathrm{Siruseri}Siruseri城中的道路都是单向的。不同的道路由路口连接。按照法律规定，在每个路口都设立了一个Siruseri\mathrm{Siruseri}Siruseri银行的ATM\mathrm{ATM}ATM取款机。令人奇怪的是，Siruseri\mathrm{Siruseri}Siruseri的酒吧都设在路口，虽然并不是每个路口都设有酒吧。
Banditiji\mathrm{Banditiji}Banditiji计划实施Siruseri\mathrm{Siruseri}Siruseri有史以来最惊天动地的ATM\mathrm{ATM}ATM抢劫。他将从市中心出发，沿着单向道路行驶，抢劫所有他途径的ATM\mathrm{ATM}ATM机，最终他将在一个酒吧庆祝他的胜利。
使用高超的黑客技术，他获知了每个ATM\mathrm{ATM}ATM机中可以掠取的现金数额。他希望你帮助他计算从市中心出发最后到达某酒吧时最多能抢劫的现金数额。他希望你帮助他计算从市中心出发最后到达某个酒吧时最多能抢劫的现金总数。他可以经过同一路口或道路任意多次。但只要他抢劫某个ATM\mathrm{ATM}ATM机后，该ATM\mathrm{ATM}ATM机就不会再有钱了。

Input

第一行包含两个整数NNN、MMM。NNN表示路口的个数，MMM表示道路条数。接下来MMM行，每行两个整数，这两个整数都在111到NNN之间，第i+1i+1i+1行的两个整数表示第i条道路的起点和终点的路口编号。接下来NNN行，每行一个整数，按顺序表示每个路口处的ATM\mathrm{ATM}ATM机中的钱数。接下来一行包含两个整数SSS、PPP，SSS表示市中心的编号，也就是出发的路口。PPP表示酒吧数目。接下来的一行中有PPP个整数，表示PPP个有酒吧的路口的编号。

Output

输出一个整数，表示Banditji\mathrm{Banditji}Banditji从市中心开始到某个酒吧结束所能抢劫的最多的现金总数。

Solution

题面有一种莫名其妙的喜感
这道题是我的第一道APIO\mathrm{APIO}APIO题（虽然是水题）
首先，很容易看出要tarjan缩点。题目说城市的路是单向边，而且可以经过任意一点任意多次，所以走到强连通分量，这个强连通分量的所有点都可以到达，所有ATM\mathrm{ATM}ATM机都可以抢。
于是我们先缩点，顺带记录每个分量的ATM\mathrm{ATM}ATM机价值总和，并统计每个分量有没有酒吧，即能否作为停止点。然后跑一遍SPFA，找以所有带酒吧的分量为终点的最长路即可。
