BZOJ3998【TJOI2015】弦论 <后缀自动机>

Posted on 2018-03-12 Symbols count in article: 687 Reading time ≈ 2 mins.

Problem

【TJOI2015】弦论

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

对于一个给定长度为NNN的字符串，求它的第KKK小子串是什么。

Input

第一行是一个仅由小写英文字母构成的字符串SSS。
第二行为两个整数TTT和KKK，TTT为000则表示不同位置的相同子串算作一个，T=1T=1T=1则表示不同位置的相同子串算作多个。KKK的意义如题所述。

Output

输出仅一行，为一个数字串，为第KKK小的子串。如果子串数目不足KKK个，则输出−1-1−1。

Sample Input

Sample Output

标签：后缀自动机

Solution

后缀自动机的模板题。

对于两个询问的找到第KKK大，预处理出从每个状态节点向后有多少种符合规则的字符串，然后每次选择走哪个字符即可。

关于预处理：

  • 对于T=0T=0T=0，即计算后缀自动机所形成的DAGDAGDAG上有多少条从初始状态出发路径，将每一个状态节点初值赋为111后按拓扑序在DAGDAGDAG上DPDPDP即可。
  • 对于T=1T=1T=1，类似上面，也是按拓扑序在DAGDAGDAG上作DPDPDP，只是此时只有表示原字符串每个前缀结束状态的节点的初值为111。
