BZOJ5336【TJOI2018】Party

Posted on 2018-07-28 Symbols count in article: 1.1k Reading time ≈ 4 mins.

Problem

【TJOI2018】Party

Time  Limit:  60  Sec\mathrm{Time\;Limit:\;60\;Sec}TimeLimit:60Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

小豆参加了NOI\mathrm{NOI}NOI的游园会，会场上每完成一个项目就会获得一个奖章，奖章只会是N,O,I的字样。在会场上他收集到了KKK个奖章组成的串。
兑奖规则是奖章串和兑奖串的最长公共子序列长度为小豆最后奖励的等级。
现在已知兑奖串长度为NNN，并且在兑奖串上不会出现连续三个奖章为NOI，即奖章中不会出现子串NOI。
现在小豆想知道各个奖励等级会对应多少个不同的合法兑奖串。

Input

第一行两个数，N,KN,KN,K分别代表兑奖串的长度，小豆收集的奖章串的长度。
第二行一共KKK个字符，表示小豆得到奖章串。

Output

一共K+1K+1K+1行，第iii行表示小豆最后奖励等级为i−1i-1i−1的不同的合法兑奖串的个数，可能这个数会很大，结果对109+710^9+7109+7取模。

Sample Output

Explanation

最长公共子序列长度000的串有：III
最长公共子序列长度222的串有：NON, NNO, NOO, ONO, INO, NIO
除去NOI，余下的19  (26−6−1)19\;(26-6-1)19(26−6−1)种为最长公共子序列长度为111。

HINT

N≤1000N\le1000N≤1000，K≤15K\le15K≤15

标签：DP套DP 状压DP

Solution

观察求最长公共子序列的DP\mathrm{DP}DP，发现在其中一个维度上，dpdpdp值是不下降的，并且相邻两者间差为000或111。由于题目中K≤15K\le15K≤15，我们可以将DP\mathrm{DP}DP数组的一行差分后进行状态压缩。
当两个串做最长公共子序列时，我们可以将二维DP\mathrm{DP}DP看作在AAA串后面每次添一个字符，然后枚举BBB串的每个位置，看新加入的字符会产生哪些影响。将每个横行的dpdpdp值差分状压后，可以当成一个一维DP\mathrm{DP}DP，只是每次的状态都是一个表示一整行的二进制数。这样枚举AAA中添哪个字符即可从一个状态转移到另一个状态。
对于”不能含子串NOI“的限制，我们只需要再多记录一维状态[0/1/2][0/1/2][0/1/2]表示当前匹配到NOI的哪个位置，若转移后匹配到第三个位置，则不能转移。
预处理状态转移trans[sta][c]trans[sta][c]trans[sta][c]表示stastasta状态上在AAA串添字符ccc能得到的状态，用f[i][j][k]f[i][j][k]f[i][j][k]表示当前填到第iii位，状态为jjj，匹配到NOI的kkk位置上，枚举添哪个字符，设最后一位转到lll，若l≠3l\ne3l=3，则f[i][j][k]f[i][j][k]f[i][j][k]对f[i+1][trans[j][c]][l]f[i+1][trans[j][c]][l]f[i+1][trans[j][c]][l]有贡献。
