Problem

【Lydsy1704月赛】抵制克苏恩

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

小Q\mathrm{小Q}小Q同学现在沉迷炉石传说不能自拔。他发现一张名为克苏恩的牌很不公平。
如果你不玩炉石传说，不必担心，小Q\mathrm{小Q}小Q同学会告诉你所有相关的细节。炉石传说是这样的一个游戏，每个玩家拥有一个303030点血量的英雄，并且可以用牌召唤至多777个随从帮助玩家攻击对手，其中每个随从也拥有自己的血量和攻击力。
小Q\mathrm{小Q}小Q同学有很多次游戏失败都是因为对手使用了克苏恩这张牌，所以他想找到一些方法来抵御克苏恩。他去求助职业炉石传说玩家椎名真白，真白告诉他使用奴隶主这张牌就可以啦。
如果你不明白我上面在说什么，不必担心，小Q\mathrm{小Q}小Q同学会告诉你他想让你做什么。
现在小Q\mathrm{小Q}小Q同学会给出克苏恩的攻击力是KKK，表示克苏恩会攻击KKK次，每次会从对方场上的英雄和随从中随机选择一个并对其产生111点伤害。
现在对方有一名克苏恩，你有一些奴隶主作为随从，每名奴隶主的血量是给定的。如果克苏恩攻击了你的一名奴隶主，那么这名奴隶主的血量会减少111点，当其血量小于等于000时会死亡，如果受到攻击后不死亡，并且你的随从数量没有达到777，这名奴隶主会召唤一个拥有000点血量的新奴隶主作为你的随从；如果克苏恩攻击了你的英雄，你的英雄会记录受到111点伤害。你应该注意到了，每当克苏恩进行一次攻击，你场上的随从可能发生很大的变化。
小Q\mathrm{小Q}小Q同学为你假设了克苏恩的攻击力，你场上分别有111点、222点、333点血量的奴隶主数量，你可以计算出你的英雄受到的总伤害的期望值是多少吗？

Input

输入包含多局游戏。
第一行包含一个整数T  (T<100)T\;(T<100)T(T<100)，表示游戏的局数。
每局游戏仅占一行，包含四个非负整数K,A,B,CK,A,B,CK,A,B,C，表示克苏恩的攻击力是KKK，你有AAA个111点血量的奴隶主，BBB个222点血量的奴隶主，CCC个333点血量的奴隶主。
保证KKK是小于505050的正数，A+B+C≤7A+B+C\le 7A+B+C≤7 。

Output

对于每局游戏，输出一个数字表示总伤害的期望值，保留两位小数。
