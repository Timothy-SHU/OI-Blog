BZOJ4514【SDOI2016】数字配对 <费用流>

Posted on 2017-12-25 Symbols count in article: 865 Reading time ≈ 3 mins.

Problem

【SDOI2016】数字配对

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

有 nnn 种数字，第 iii 种数字是 aia_iai​、有 bib_ibi​ 个，权值是 cic_ici​。
若两个数字 ai, aja_i,\ a_jai​, aj​ 满足，aia_iai​ 是 aja_jaj​ 的倍数，且 aiaj\frac{a_i}{a_j}aj​ai​​ 是一个质数，那么这两个数字可以配对，并获得 ci×cjc_i\times c_jci​×cj​ 的价值。
一个数字只能参与一次配对，可以不参与配对。
在获得的价值总和不小于 000 的前提下，求最多进行多少次配对。

Input

第一行一个整数 nnn。
第二行 nnn 个整数 a1,a2,⋯ ,ana_1,a_2,\cdots,a_na1​,a2​,⋯,an​。
第三行 nnn 个整数b1,b2,⋯ ,bnb_1,b_2,\cdots,b_nb1​,b2​,⋯,bn​。
第四行 nnn 个整数 c1,c2,⋯ ,cnc_1,c_2,\cdots,c_nc1​,c2​,⋯,cn​。

Output

一行一个数，最多进行多少次配对

HINT

n≤200，ai≤109，bi≤105，∣ci∣≤105n\le 200，a_i\le 10^9，b_i\le 10^5，∣c_i∣\le 10^5n≤200，ai​≤109，bi​≤105，∣ci​∣≤105

标签：费用流

Solution

比较难想的二分图建模。
记f(x)f(x)f(x)为xxx分解质因数后的项数（4=2×24=2\times 24=2×2算两项），aiaj\frac{a_i}{a_j}aj​ai​​为质数的充要条件为f(x)=f(y)+1且y∣xf(x)=f(y)+1且y|xf(x)=f(y)+1且y∣x。
可知f(x)f(x)f(x)奇偶性相同的一定不能配对，所以可建二分图，f(x)f(x)f(x)为奇在一边，f(x)f(x)f(x)为偶在另一边。
两边间连边则看是否有整除条件，流量为INFINFINF，费用为ci×cjc_i\times c_jci​×cj​。
把每个点拆成左右两个点，左边与源点相连，右边与汇点相连，容量为 bib_ibi​，费用为零。
跑费用流每增广一次就判一下是否费用小于 000，注意最后退出的时候不要把新加入的流全退完，退一部分至刚好大于等于000即可。
