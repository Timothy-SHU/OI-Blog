BZOJ1857【SCOI2010】传送带 <三分法>

Posted on 2018-03-20 Symbols count in article: 779 Reading time ≈ 3 mins.

Problem

【SCOI2010】传送带

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  64  MB\mathrm{Memory\;Limit:\;64\;MB}MemoryLimit:64MB

Description

在一个222维平面上有两条传送带，每一条传送带可以看成是一条线段。两条传送带分别为线段ABABAB和线段CDCDCD。
lxhgww\mathrm{lxhgww}lxhgww在ABABAB上的移动速度为PPP，在CDCDCD上的移动速度为QQQ，在平面上的移动速度RRR。
现在lxhgww\mathrm{lxhgww}lxhgww想从AAA点走到DDD点，他想知道最少需要走多长时间。

Input

输入数据第一行是444个整数，表示AAA和BBB的坐标，分别为Ax,Ay,Bx,ByA_x,A_y,B_x,B_yAx​,Ay​,Bx​,By​。
第二行是444个整数，表示CCC和DDD的坐标，分别为Cx,Cy,Dx,DyC_x,C_y,D_x,D_yCx​,Cy​,Dx​,Dy​。
第三行是333个整数，分别是P,Q,RP,Q,RP,Q,R。

Output

输出数据为一行，表示lxhgww\mathrm{lxhgww}lxhgww从AAA点走到DDD点的最短时间，保留到小数点后222位。

Hint

对于100%100\%100%的数据，1≤Ax,Ay,Bx,By,Cx,Cy,Dx,Dy≤1031\le A_x,A_y,B_x,B_y,C_x,C_y,D_x,D_y\le10^31≤Ax​,Ay​,Bx​,By​,Cx​,Cy​,Dx​,Dy​≤103，1≤P,Q,R≤101\le P,Q,R\le101≤P,Q,R≤10。

标签：三分法

Solution

三分套三分，据说有大学物理结论可以O(1)\mathrm{O(1)}O(1)做…
首先一定是从AAA走到ABABAB上的一点EEE，然后走平面到CDCDCD上的一点FFF，最后从FFF沿CDCDCD走到DDD。
考虑确定ABABAB上的EEE点，那么若将FFF点的所有位置都计算答案并画成函数图像，一定是一个单峰函数。那么确定EEE后可以三分法找到最佳FFF位置。
同样地，对于每个EEE位置的最佳答案，也会呈一个单峰函数。因此可以先对EEE在ABABAB上进行三分，每次三分到的两个端点需要再对FFF在CDCDCD上进行三分找到最小时间。
这样一来，可以先在ABABAB上三分EEE，每次三分找端点最值的时候再在CDCDCD上三分FFF，三分套三分即可找到答案。
