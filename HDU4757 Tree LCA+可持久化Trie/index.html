    HDU4757 Tree < LCA+可持久化Trie >

    Posted on 2017-10-17 Symbols count in article: 998 Reading time ≈ 4 mins.

    Problem

    Tree

    Description

    Zero and One are good friends who always have fun with each other.
    This time, they decide to do something on a tree which is a kind of graph that there is only one path from node to node. First, Zero will give One an tree and every node in this tree has a value. Then, Zero will ask One a series of queries. Each query contains three parameters: xxx, yyy, zzz which mean that he want to know the maximum value produced by zzz xorxorxor each value on the path from node xxx to node yyy (include node xxx, node yyy). Unfortunately, One has no idea in this question. So he need you to solve it.

    Input

    There are several test cases and the cases end with EOF\mathrm{EOF}EOF. For each case:
    The first line contains two integers n(1≤n≤105)n(1\le n\le 10^5)n(1≤n≤105) and m(1≤m≤105)m(1\le m\le 10^5)m(1≤m≤105), which are the amount of tree’s nodes and queries, respectively.
    The second line contains nnn integers a[1..n]a[1..n]a[1..n] and a[i]  (0≤a[i]<216)a[i]\;(0\le a[i]<2^{16})a[i](0≤a[i]<216) is the value on the ithi^{th}ith node.
    The next n–1n–1n–1 lines contains two integers uuu vvv, which means there is an connection between uuu and vvv.
    The next m lines contains three integers xxx yyy zzz, which are the parameters of Zero’s query.

    Output

    For each query, output the answer.

    Sample Output

    Translation

    给出一棵树，求两点x,yx,yx,y间树上链路中的数异或zzz得到的最大结果。

    标签：LCA 可持久化Trie

    Solution

    最大异或和上树…
    有异或，一个经典的解法就是建一棵010101字典树，然后贪心跑一遍，尽量选和给出数当前位不同的数。
    对于树上的此类问题，可以把Trie树可持久化，对于每个点，存它到根节点的路径上所有数的Trie树，这样是有前缀和性质的，求LCA\mathrm{LCA}LCA后作差即可求出链路上的数。
    可持久化方法和主席树差不多。
