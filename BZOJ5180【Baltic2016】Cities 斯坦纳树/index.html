BZOJ5180【Baltic2016】Cities <斯坦纳树>

Posted on 2018-04-17 Symbols count in article: 691 Reading time ≈ 3 mins.

Problem

【Baltic2016】Cities

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

给定nnn个点，mmm条双向边的图，其中有kkk个点是重要的，每条边都有一定的长度。
现在要你选定一些边来构成一个图，要使得kkk个重要的点相互连通，求边的长度和的最小值。

Input

共m+2m+2m+2行
第111行读入n,k,mn,k,mn,k,m，表示nnn个点，kkk个重要的点，mmm条边
第222行读入kkk个重要点的编号
第333至第m+2m+2m+2行，每行包括333个数字a,b,ca,b,ca,b,c，表示有一条从aaa到bbb长度为ccc的双向路径

Output

共111行，即最小长度和

Sample Output

HINT

k≤5,  n≤105,  1≤m≤2×105k\le5,\;n\le10^5,\;1\le m\le2\times10^5k≤5,n≤105,1≤m≤2×105

标签：斯坦纳树 状压DP

Solution

斯坦纳树裸题。

斯坦纳树的基本解法是状压DP\mathrm{DP}DP，压缩联通状态进行DP\mathrm{DP}DP，f[s][i]f[s][i]f[s][i]表示在iii点，联通状态为sss的最小花费。
有两种转移：

  • 状态sss可以通过两个状态组合而来，对于sss的一个子集ttt，有f[s][i]=max⁡(f[s][i],f[t][i]+f[s−t][i])f[s][i]=\max(f[s][i],f[t][i]+f[s-t][i])f[s][i]=max(f[s][i],f[t][i]+f[s−t][i])
  • 状态sss也可以在同层向邻接点扩展，即最短路中的松弛操作，对于边u,vu,vu,v，有f[s][v]=max⁡(f[s][v],f[s][u]+Edgeu,v)f[s][v]=\max(f[s][v],f[s][u]+Edge_{u,v})f[s][v]=max(f[s][v],f[s][u]+Edgeu,v​)，可以跑最短路更新。

此题有点卡，注意不要用SPFA，要用堆优Dijkstra。
