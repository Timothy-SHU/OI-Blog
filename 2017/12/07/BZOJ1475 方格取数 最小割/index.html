BZOJ1475 方格取数 <黑白染色+最小割>

Posted on 2017-12-07

Problem

方格取数

Time Limit: $5 Sec$
Memory Limit: $64 MB$

Description

在一个$n\times n$的方格里，每个格子里都有一个正整数。从中取出若干数，使得任意两个取出的数所在格子没有公共边，且取出的数的总和尽量大。

Input

第一行一个数$n$（$n\le 30$），接下来$n$行每行$n$个数描述一个方阵

Output

仅一个数，即最大和

Sample Input

Sample Output

标签：黑白染色 最小割

Solution

将网格图黑白染色，同色格子间不会冲突。
建模：$S\to 黑格\ \ 容量val$，$白格\to T\ \ 容量val$，$黑格\to 白格\ \ 容量\infty$
$ans = 总权值-最小割$
