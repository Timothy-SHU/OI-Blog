BZOJ1221【HNOI2001】软件开发 <拆点费用流>

Posted on 2017-12-26

Problem

【HNOI2001】软件开发

Time Limit: 10Sec10 Sec10Sec
Memory Limit: 162MB162 MB162MB

Description

某软件公司正在规划一项nnn天的软件开发计划，根据开发计划第iii天需要nin_ini​个软件开发人员，为了提高软件开发人员的效率，公司给软件人员提供了很多的服务，其中一项服务就是要为每个开发人员每天提供一块消毒毛巾，这种消毒毛巾使用一天后必须再做消毒处理后才能使用。消毒方式有两种，AAA种方式的消毒需要aaa天时间，BBB种方式的消毒需要bbb天（b>ab>ab>a），AAA种消毒方式的费用为每块毛巾fAf_AfA​, BBB种消毒方式的费用为每块毛巾fBf_BfB​，而买一块新毛巾的费用为fff（新毛巾是已消毒的，当天可以使用）；而且f>fA>fBf>f_A>f_Bf>fA​>fB​。公司经理正在规划在这nnn天中，每天买多少块新毛巾、每天送多少块毛巾进行AAA种消毒和每天送多少块毛巾进行BBB种消毒。当然，公司经理希望费用最低。你的任务就是：为该软件公司计划每天买多少块毛巾、每天多少块毛巾进行AAA种消毒和多少毛巾进行BBB种消毒，使公司在这项nnn天的软件开发中，提供毛巾服务的总费用最低。

Input

第111行为n,a,b,f,fA,fBn,a,b,f,f_A,f_Bn,a,b,f,fA​,fB​. 第222行为n1,n2,⋯ ,nnn_1,n_2,\cdots,n_nn1​,n2​,⋯,nn​. （注：1≤f,fA,fB≤60,1≤n≤10001\le f,f_A,f_B\le 60,1\le n\le 10001≤f,fA​,fB​≤60,1≤n≤1000）

Output

最少费用

Solution

拆点费用流套路题
把每天拆成两个点XiYi\frac{X_i}{Y_i}Yi​Xi​​，表示用过的毛巾和新洗好的毛巾。
建图：
S→XiS\to X_iS→Xi​ 容量nin_ini​ 费用000
Yi→TY_i\to TYi​→T 容量nin_ini​ 费用000
S→YiS\to Y_iS→Yi​ 容量∞\infty∞ 费用fff
Xi→Xi+1X_i\to X_i+1Xi​→Xi​+1 容量∞\infty∞ 费用000
Xi→Yi+a+1X_i\to Y_i+a+1Xi​→Yi​+a+1 容量∞\infty∞ 费用fAf_AfA​
Xi→Yi+b+1X_i\to Y_i+b+1Xi​→Yi​+b+1 容量∞\infty∞ 费用fBf_BfB​
