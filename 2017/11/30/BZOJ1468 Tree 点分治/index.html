BZOJ1468 Tree <点分治>

Posted on 2017-11-30

Problem

Tree

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  64  MB\mathrm{Memory\;Limit:\;64\;MB}MemoryLimit:64MB

Description

给你一棵树以及这棵树上边的距离.问有多少对点它们两者间的距离小于等于KKK

Input

NNN（N≤40000N\le 40000N≤40000） 接下来n−1n-1n−1行边描述管道，按照题目中写的输入 接下来是KKK

Output

一行，有多少对点之间的距离小于等于KKK

Sample Input

Sample Output

标签：点分治

Solution

点分治板题。
对于每个点，我们计算穿过它的路径的条数。
这个值等于它的子树中所有深度之和小于kkk的点对数，可以双指针搞一搞。
但这样两个在同一儿子内的点对会算重，所以需要减去它的每个儿子的子树中深度之和小于k−ck-ck−c的点对数，其中ccc为它到此儿子的距离。这也可以双指针搞一搞。
对于每次分治，我们都要找到当前子树的重心，以保证复杂度为O(nlog⁡(n))O(n\log(n))O(nlog(n))。
