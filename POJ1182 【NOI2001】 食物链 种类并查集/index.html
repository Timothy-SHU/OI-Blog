POJ1182 【NOI2001】 食物链 <种类并查集>

Posted on 2017-08-17 Symbols count in article: 869 Reading time ≈ 3 mins.

Problem

食物链

Description

动物王国中有三类动物A,B,CA,B,CA,B,C，这三类动物的食物链构成了有趣的环形。AAA吃BBB，BBB吃CCC，CCC吃AAA。
有NNN个动物，以1∼N1\sim N1∼N编号。每个动物都是A,B,CA,B,CA,B,C中的一种，但是我们并不知道它到底是哪一种。
人用两种说法对这NNN个动物所构成的食物链关系进行描述：
一种说法是1  X  Y1\;X\;Y1XY，表示XXX和YYY是同类。
二种说法是2  X  Y2\;X\;Y2XY，表示XXX吃YYY。
人对NNN个动物，用上述两种说法，一句接一句地说出KKK句话，这KKK句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

  • 当前的话与前面的某些真的话冲突，就是假话；
  • 当前的话中XXX或YYY比NNN大，就是假话；
  • 当前的话表示XXX吃XXX，就是假话。

你的任务是根据给定的N（1≤N≤50000）N（1 \le N \le 50000）N（1≤N≤50000）和KKK句话（0≤K≤100000）（0 \le K \le 100000）（0≤K≤100000），输出假话的总数。

Input

第一行是两个整数NNN和KKK，以一个空格分隔。
以下KKK行每行是三个正整数 DDD，XXX，YYY，两数之间用一个空格隔开，其中DDD表示说法的种类。
若D=1D=1D=1，则表示XXX和YYY是同类。
若D=2D=2D=2，则表示XXX吃YYY。

Output

只有一个整数，表示假话的数目。

Solution

逻辑推理的题有一部分和并查集有关，此题是种类并查集的经典例题。
首先我们把每个动物分成三个点，对于点iii，点iii表示第iii个动物的种类，点i+ni+ni+n表示第iii个动物的食物，点i+2ni+2ni+2n表示第iii个动物的天敌。
这样一来，提供信息：xxx和yyy同类，相当于提供三条信息：

  1. xxx、yyy在同一个集中
  2. x+nx+nx+n、y+ny+ny+n在同一个集中
  3. x+2nx+2nx+2n、y+2ny+2ny+2n在同一个集中

于是我们merge(x,y),merge(x+n,y+n),merge(x+2n,y+2n)merge(x, y), merge(x+n, y+n), merge(x+2n, y+2n)merge(x,y),merge(x+n,y+n),merge(x+2n,y+2n)
同理，提供信息：xxx吃yyy，相当于提供三条信息：

  1. xxx、y+2ny+2ny+2n在同一个集中

  2. x+nx+nx+n、yyy在同一个集中

  3. x+2nx+2nx+2n、y+ny+ny+n在同一个集中

于是我们merge(x,y+2n),merge(x+n,y),merge(x+2n,y+n)merge(x, y+2n), merge(x+n, y), merge(x+2n, y+n)merge(x,y+2n),merge(x+n,y),merge(x+2n,y+n)
如果在接到信息xxx和yyy同类后，发现xxx和y+ny+ny+n同类或xxx和y+2ny+2ny+2n同类，则此信息与先前信息矛盾。因为对称性，我们不用再判断yyy是否和x+nx+nx+n或x+2nx+2nx+2n同类。同理可处理xxx吃yyy的情况。
