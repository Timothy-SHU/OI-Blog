Description

神犇SJY\mathrm{SJY}SJY虐完HEOI\mathrm{HEOI}HEOI之后给傻XLYD\mathrm{LYD}LYD出了一题：
SJY\mathrm{SJY}SJY是T国的公主，平时的一大爱好是作诗。由于时间紧迫，SHY\mathrm{SHY}SHY作完诗之后还要虐OI\mathrm{OI}OI，于是SHY\mathrm{SHY}SHY找来一篇长度为NNN的文章，阅读MMM次，每次只阅读其中连续的一段[l,r][l,r][l,r]，从这一段中选出一些汉字构成诗。
因为SHY\mathrm{SHY}SHY喜欢对偶，所以SHY\mathrm{SHY}SHY规定最后选出的每个汉字都必须在[l,r][l,r][l,r]里出现了正偶数次。而且SHY\mathrm{SHY}SHY认为选出的汉字的种类数（两个一样的汉字称为同一种）越多越好（为了拿到更多的素材！）。于是SHY\mathrm{SHY}SHY请LYD\mathrm{LYD}LYD安排选法。LYD\mathrm{LYD}LYD这种傻X当然不会了，于是向你请教……
问题简述：NNN个数，MMM组询问，每次问[l,r][l,r][l,r]中有多少个数出现正偶数次。

Solution

由于”出现次数为偶数“不好维护，需要分块。
将序列分为n\sqrt{n}n​个块，预处理出前iii块内每个数的出现次数，以及两块间的有多少个数出现偶数次。对于每个询问，将左右端点所在块中间的块有多少个数出现偶数次作为基础答案，然后暴力枚举块外的数，结合预处理出的“前iii块内每个数的出现次数”判断其对答案的影响即可。
