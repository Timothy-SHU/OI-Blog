Now, Birzhan has to answer qqq queries each of them consisting of two integers lll and rrr (1≤l≤r≤n1\le l\le r\le n1≤l≤r≤n). We define f(l,r)f(l,r)f(l,r) as the sum of the squares of every integer from 111 to mmm if it exists on the visible side of any card numbered from lll to rrr. Problem

Number Table

Time  Limit:  2  seconds\mathrm{Time\;Limit:\;2\;seconds}TimeLimit:2seconds
Memory  Limit:  216  megabytes\mathrm{Memory\;Limit:\;216\;megabytes}MemoryLimit:216megabytes

Description

As it has been found out recently, all the Berland’s current economical state can be described using a simple table n×mn\times mn×m in size. nnn ― the number of days in each Berland month, mmm ― the number of months. Thus, a table cell corresponds to a day and a month of the Berland’s year. Each cell will contain either 111, or −1-1−1, which means the state’s gains in a particular month, on a particular day. 111 corresponds to profits, −1-1−1 corresponds to losses. It turned out important for successful development to analyze the data on the state of the economy of the previous year, however when the treasurers referred to the archives to retrieve the data, it turned out that the table had been substantially damaged. In some table cells the number values had faded and were impossible to be deciphered. It is known that the number of cells in which the data had been preserved is strictly less than max⁡(n,m)\max(n,m)max(n,m). However, there is additional information ― the product of the numbers in each line and column equaled −1-1−1. Your task is to find out how many different tables may conform to the preserved data. As the answer to the task can be quite large, you have to find it modulo ppp.

Input

The first line contains integers nnn and mmm (1≤n,m≤1000)(1\le n,m\le1000)(1≤n,m≤1000). The second line contains the integer kkk (0≤k<max⁡(n,m))(0\le k<\max(n,m))(0≤k<max(n,m)) ― the number of cells in which the data had been preserved. The next kkk lines contain the data on the state of the table in the preserved cells. Each line is of the form a b c, where aaa (1≤a≤n)(1\le a\le n)(1≤a≤n) ― the number of the table row, bbb (1≤b≤m)(1\le b\le m)(1≤b≤m) ― the number of the column, ccc ― the value containing in the cell (111 or −1-1−1). They are numbered starting from 111. It is guaranteed that no two lines with same aaa and bbb values exist. The last line contains an integer ppp (2≤p≤109+7)(2\le p\le10^9+7)(2≤p≤109+7).
