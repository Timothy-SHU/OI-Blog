POJ1201 Intervals <差分约束系统>

Posted on 2017-08-15 Symbols count in article: 761 Reading time ≈ 3 mins.

Problem

Intervals

Description

You are given nnn closed, integer intervals [ai,bi][a_i, b_i][ai​,bi​] and nnn integers c1∼cnc_1\sim c_nc1​∼cn​.
Write a program that:
reads the number of intervals, their end points and integers c1∼cnc_1\sim c_nc1​∼cn​ from the standard input, computes the minimal size of a set ZZZ of integers which has at least cic_ici​ common elements with interval [ai,bi][a_i, b_i][ai​,bi​], for each i=1,2,⋯ ,ni=1,2,\cdots ,ni=1,2,⋯,n, writes the answer to the standard output.

Input

The first line of the input contains an integer nnn (1≤n≤5×1041\le n\le 5\times 10^41≤n≤5×104) – the number of intervals. The following n lines describe the intervals. The (i+1)th(i+1)^{th}(i+1)th line of the input contains three integers aia_iai​, bib_ibi​ and cic_ici​ separated by single spaces and such that 0≤ai≤bi≤5×1040\le a_i\le b_i\le 5\times 10^40≤ai​≤bi​≤5×104 and 1≤ci≤bi−ai+11\le c_i\le b_i - a_i+11≤ci​≤bi​−ai​+1.

Output

The output contains exactly one integer equal to the minimal size of set ZZZ sharing at least cic_ici​ elements with interval [ai,bi][a_i, b_i][ai​,bi​], for each i=1,2,⋯ ,ni=1,2,\cdots ,ni=1,2,⋯,n.

Translation

给出nnn个区间，试确定一个集合使得对于i=1∼ni=1\sim ni=1∼n，第iii个区间内至少有cic_ici​个数，并使得此集合尽量小，输出最小大小。

Solution

首先用前缀和。sum[i]sum[i]sum[i]表示从111到iii中共选出多少个数到集合中。这样对于集合[ai,bi][a_i,b_i][ai​,bi​]，有sum[bi]−sum[ai−1]≥cisum[b_i]-sum[a_i-1]\ge c_isum[bi​]−sum[ai​−1]≥ci​，于是我们可以从点ai−1a_i-1ai​−1到bib_ibi​连一条权值为cic_ici​的边。因为题意是要满足所有的边，所以我们需要找最长路。
此题有一些细节问题。首先，找最长路需要起点和终点，我们需要找到这些集合覆盖的范围，即找到左端点（其实应该是左端点−1-1−1）的最小值sss和右端点的最大值ttt，找sss到ttt的最大值。此外，光有上述的那些边时无法构成一个连通图的，所以我们需要找一些隐含条件。可以发现有sum[i]−sum[i−1]≥0sum[i]-sum[i-1]\ge 0sum[i]−sum[i−1]≥0，sum[i]−sum[i−1]≤1sum[i]-sum[i-1]\le 1sum[i]−sum[i−1]≤1，为了保持一致，应将后面的式子转化为大于等于，即sum[i−1]−sum[i]≥−1sum[i-1]-sum[i]\ge -1sum[i−1]−sum[i]≥−1，这样对于i=s+1∼ti=s+1\sim ti=s+1∼t，从i−1i-1i−1到iii连接一条权值为000的路，从iii到i−1i-1i−1连接一条权值为−1-1−1的路，之后就可以直接用SPFASPFASPFA找最长路了。
