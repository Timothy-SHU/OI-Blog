BZOJ2038 小Z的袜子 <莫队>

Posted on 2017-09-25 Symbols count in article: 991 Reading time ≈ 4 mins.

Problem

小Z的袜子

Description

作为一个生活散漫的人，小ZZZ每天早上都要耗费很久从一堆五颜六色的袜子中找出一双来穿。终于有一天，小ZZZ再也无法忍受这恼人的找袜子过程，于是他决定听天由命……
具体来说，小ZZZ把这NNN只袜子从111到NNN编号，然后从编号LLL到RRR(尽管小ZZZ并不在意两只袜子是不是完整的一双，甚至不在意两只袜子是否一左一右，他却很在意袜子的颜色，毕竟穿两只不同色的袜子会很尴尬)。
你的任务便是告诉小ZZZ，他有多大的概率抽到两只颜色相同的袜子。当然，小ZZZ希望这个概率尽量高，所以他可能会询问多个(L,R)(L,R)(L,R)以方便自己选择。

Input

输入文件第一行包含两个正整数NNN和MMM。NNN为袜子的数量，MMM为小ZZZ所提的询问的数量。接下来一行包含NNN个正整数CiC_iCi​，其中CiC_iCi​表示第iii只袜子的颜色，相同的颜色用相同的数字表示。再接下来MMM行，每行两个正整数LLL，RRR表示一个询问。

Output

包含MMM行，对于每个询问在一行中输出分数A/BA/BA/B表示从该询问的区间[L,R][L,R][L,R]中随机抽出两只袜子颜色相同的概率。若该概率为000则输出0/10/10/1，否则输出的A/BA/BA/B必须为最简分数。

Hint

样例解释
询问111：共C52=10C_5^2=10C52​=10种可能，其中抽出两个222有111种可能，抽出两个333有333种可能，概率为(1+3)/10=4/10=2/5(1+3)/10=4/10=2/5(1+3)/10=4/10=2/5。
询问222：共C32=3C_3^2=3C32​=3种可能，无法抽到颜色相同的袜子，概率为0/3=0/10/3=0/10/3=0/1。
询问333：共C32=3C_3^2=3C32​=3种可能，均为抽出两个3，概率为3/3=1/13/3=1/13/3=1/1。
