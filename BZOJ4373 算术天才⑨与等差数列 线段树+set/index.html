BZOJ4373 算术天才⑨与等差数列 <线段树+set>

Posted on 2018-09-10 Symbols count in article: 1.3k Reading time ≈ 5 mins.

Problem

算术天才⑨与等差数列

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

算术天才⑨非常喜欢和等差数列玩耍。
有一天，他给了你一个长度为nnn的序列，其中第iii个数为aia_iai​。
他想考考你，每次他会给出询问l,r,kl,r,kl,r,k，问区间[l,r][l,r][l,r]内的数从小到大排序后能否形成公差为kkk的等差数列。
当然，他还会不断修改其中的某一项。
为了不被他鄙视，你必须要快速并正确地回答完所有问题。
注意：只有一个数的数列也是等差数列。

Input

第一行包含两个正整数n,m  (1≤n,m≤300000)n,m\;(1\le n,m\le300000)n,m(1≤n,m≤300000)，分别表示序列的长度和操作的次数。
第二行包含nnn个整数，依次表示序列中的每个数ai  (0≤ai≤109)a_i\;(0\le a_i\le10^9)ai​(0≤ai​≤109)。
接下来mmm行，每行一开始为一个数opopop，
若op=1op=1op=1，则接下来两个整数x,y  (1≤x≤n,0≤y≤109)x,y\;(1\le x\le n,0\le y\le10^9)x,y(1≤x≤n,0≤y≤109)，表示把axa_xax​修改为yyy。
若op=2op=2op=2，则接下来三个整数l,r,k  (1≤l≤r≤n,0≤k≤109)l,r,k\;(1\le l\le r\le n,0\le k\le10^9)l,r,k(1≤l≤r≤n,0≤k≤109)，表示一个询问。
在本题中，x,y,l,r,kx,y,l,r,kx,y,l,r,k都是经过加密的，都需要异或你之前输出的Yes的个数来进行解密。

Output

输出若干行，对于每个询问，如果可以形成等差数列，那么输出Yes，否则输出No。

Solution

线段树判断等差数列，单点修改。

若区间al∼ara_l\sim a_ral​∼ar​组成等差数列，则一定有：

  • max⁡i=lrai−min⁡i=lrai=(r−l)×k\max_{i=l}^{r}a_i-\min_{i=l}^{r}a_i=(r-l)\times kmaxi=lr​ai​−mini=lr​ai​=(r−l)×k
  • 没有重复的数
  • gcd⁡{al,al+1,⋯ ,ar}=k\gcd\lbrace a_l,a_{l+1},\cdots,a_{r}\rbrace=kgcd{al​,al+1​,⋯,ar​}=k

其中若存在l=rl=rl=r的情况要单独处理。

对于第一、三个条件，用线段树存储区间min⁡,max⁡,gcd⁡\min,\max,\gcdmin,max,gcd即可。
对于第二个条件，需要在外面用set\mathrm{set}set维护每个位置的值上一次出现在哪个位置，将这个位置存入线段树中，区间求max⁡\maxmax，若最大值小于lll，则一定没有重复的数。

注意修改时不止当前位置的前驱会发生变化，所以需要多次更新。
