BZOJ3632 外太空旅行 <随机化>

Posted on 2017-11-29 Symbols count in article: 649 Reading time ≈ 2 mins.

Problem

外太空旅行

Time Limit: 5Sec5 Sec5Sec
Memory Limit: 128MB128 MB128MB

Description

在人类的触角伸向银河系的边缘之际，普通人上太空旅行已经变得稀松平常了。某理科试验班有nnn个人，现在班主任要从中选出尽量多的人去参加一次太空旅行活动。
可是nnn名同学并不是和平相处的。有的人，比如小AAA和小BBB整天狼狈为奸，是好朋友；但还有的人，比如杜鲁门和赫鲁晓夫就水火不相容。这nnn名同学，由于是理科生，都非常的理性，所以“朋友的朋友就是朋友”和“敌人的朋友就是敌人”这两句话对这些同学无效。换句话说，有可能小AAA和小BBB是朋友，小BBB和小CCC是朋友，但是小AAA和小CCC两人势如水火。
任意两个人之间要不就是敌人，要不就是朋友。
因为在太空船上发生人员斗殴事件是很恶劣也很危险的，因此选出来参加旅行活动的同学必须互相之间都是朋友。你的任务就是确定最多可以选多少人参加旅行。

Input

第一行一个整数nnn(1≤n≤501\le n\le 501≤n≤50)。所有的同学按照1∼n1\sim n1∼n编号。
接下来若干行，每行两个用空格隔开的整数a,ba, ba,b (1≤a,b≤n1\le a,b\le n1≤a,b≤n)，表示aaa和bbb是朋友。
注意：如果一个数对(x,y)(x,y)(x,y)或者(y,x)(y,x)(y,x)没有在文件中出现，那么编号为xxx和yyy的两个同学就是敌人。

Output

仅仅一个数，即最多可以选多少人参加活动。

Sample Input

Sample Output

说明：选编号为1,2,31,2,31,2,3的同学参加，他们互相都是朋友。

标签：最大团 随机化

Solution

一道随机化最大团的裸题。
每次随机出一个1∼n1\sim n1∼n的乱序排列，按照排列顺序贪心，这样随机10510^5105次取最大值即可。
