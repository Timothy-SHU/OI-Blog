Problem

【CERC2007】Robotic Sort

Time  Limit:  5  Sec\mathrm{Time\;Limit:\;5\;Sec}TimeLimit:5Sec
Memory  Limit:  64  MB\mathrm{Memory\;Limit:\;64\;MB}MemoryLimit:64MB

Description

Input

输入共两行，第一行为一个整数NNN，NNN表示物品的个数，1≤N≤1051\le N\le10^51≤N≤105。
第二行为NNN个用空格隔开的正整数，表示NNN个物品最初排列的编号。

Output

输出共一行，NNN个用空格隔开的正整数P1,P2,P3⋯PnP_1,P_2,P_3\cdots P_nP1​,P2​,P3​⋯Pn​，PiP_iPi​表示第iii次操作前第iii小的物品所在的位置。
注意：如果第iii次操作前，第iii小的物品己经在正确的位置PiP_iPi​上，我们将区间[Pi,Pi][P_i,P_i][Pi​,Pi​]反转(单个物品)。

Sample Input

Sample Output

标签：Splay

Solution

Splay\mathrm{Splay}Splay基础题。

将所有物品按照位置从小到大插入Splay\mathrm{Splay}Splay中，并维护每个结点及其子树中的最小值和最小值所在的结点指针。对于第iii次操作，在区间[i,n][i,n][i,n]找出最小值所在的结点指针，查询其位置是多少，令其为ppp，输出后反转区间[i,p][i,p][i,p]即可。

注意这道题的物品大小有重复，需要先按照大小为第一关键字，位置为第二关键字离散化后再开始操作。
