BZOJ3676【APIO2014】回文串 <回文自动机>

Posted on 2018-02-06 Symbols count in article: 557 Reading time ≈ 2 mins.

Problem

BZOJ3676【APIO2014】回文串

Time Limit: 20Sec20 Sec20Sec
Memory Limit: 128MB128 MB128MB

Description

给你一个由小写拉丁字母组成的字符串 sss。我们定义 sss 的一个子串的存在值为这个子串在 sss 中出现的次数乘以这个子串的长度。
对于给你的这个字符串 sss，求所有回文子串中的最大存在值。

Input

输入只有一行，为一个只包含小写字母(a−z)(a-z)(a−z)的非空字符串 sss。

Output

输出一个整数，表示所有回文子串中的最大存在值。

Sample Input

Input 111

Input 222

Sample Output

Output 111

Output 222

HINT

一个串是回文的，当且仅当它从左到右读和从右到左读完全一样。
在第一个样例中，回文子串有777个：aaa，bbb，ccc，abaabaaba，acaacaaca，bacabbacabbacab，abacabaabacabaabacaba，其中：

  • a出现444次，其出现值为444：4×1=44\times 1=44×1=4
  • b出现222次，其出现值为222：2×1=22\times 1=22×1=2
  • c出现111次，其出现值为111：1×1=11\times 1=11×1=1
  • aba出现222次，其出现值为666：2×3=62\times 3=62×3=6
  • aca出现111次，其出现值为333：1×3=31\times 3=31×3=3
  • bacab出现111次，其出现值为555：1×5=51\times 5=51×5=5
  • abacaba出现111次，其出现值为777：1×7=71\times 7=71×7=7

故最大回文子串出现值为777。
数据规模与评分
数据满足 1≤字符串长度≤3×1051\le 字符串长度\le 3\times 10^51≤字符串长度≤3×105。
