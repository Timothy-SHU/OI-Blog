Problem

【BJ2011集训】星器

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

Magic  Land\mathrm{Magic\;Land}MagicLand上的时间又过了若干世纪……
现在，人们谈论着一个传说：从前，他们的祖先来到了一个位于东方的岛屿，那里简直就是另外一个世界。善于分析与构造的Magic  Land\mathrm{Magic\;Land}MagicLand上的人们总是不明白那里的人们是如何不借助精确的实验与计算驱动和操纵魔法。
偶然地，一个魔法使（Magician\mathrm{Magician}Magician）来到了Magic  Land\mathrm{Magic\;Land}MagicLand，在临走的时候留下了一个神奇的盒子，叫做星器（casket  of  star\mathrm{casket\;of\;star}casketofstar）。
虽然不知道这个盒子是做什么的，但是经过了大量的实验和计算后，人们已经清楚它的一些事实：

  • 星器之中有N×MN\times MN×M个区域，可看作分成NNN行和MMM列的格子，每个区域之中有若干单位的称为“星”的对象，这个对象的最小单位已经被确定，所以，这个数量总是整数。
  • 魔法使可以驱动星器中位于同一行或同一列的不相邻（有公共边的区域称为相邻的）两个区域中各1单位的“星”，使得它们分别向中心移动1格。
  • 每一次使用2中的方法驱动“星”，将会产生魔力，魔法使会得到这一部分魔力。魔力的量等于这个两个区域之间所间隔的区域数。
    这样，我们可以用一个N×MN\times MN×M的数表来表示星器的状态，比如N=2N=2N=2，M=3M=3M=3时：
![](https://i.loli.net/2018/02/13/5a829d1a6f7f2.png)

当星器为左图的状态时，通过操纵第一行的第111和333个区域中的“星”（加粗的数字对应的区域），变为右图所示的状态，同时，将产生111单位的魔力（因为这两个区域之间恰好隔了111个区域）。
在经过了进一步的研究之后，人们知道了这个星器最初的状态（Ini\mathrm{Ini}Ini）以及最终被他们得到时的状态（Fin\mathrm{Fin}Fin）。
你希望知道，星器最多帮助它的拥有者提供了多少的魔力。即：经过一系列上述操作由初态（Ini\mathrm{Ini}Ini）变为终态（Fin\mathrm{Fin}Fin），至多产生多少魔力。
需要注意的是，显然操作过程中每个区域内“星”的数量不能是负的，即：如果那个区域已经没有“星”了，当然就不能继续操作了。

Input

第一行包含两个正整数NNN、MMM表示星器的大小。
接下来的NNN行，每行包含MMM个自然数：Inii,jIni_{i,j}Inii,j​，描绘了初态（IniIniIni）。
在一个空行后的NNN行，每行包含MMM个自然数：Fini,jFin_{i,j}Fini,j​，描绘了终态（FinFinFin）。

Output

输出一个正整数，表示至多产生的魔力。

Explanation
唯一的一种操作方法是：
对第555列的两个“星”进行一次操作，产生魔力222；
对第111列的两个“星”进行两次操作，产生魔力3+13+13+1；
对第444行的两个“星”进行一次操作，产生魔力111；
一共产生777单位的魔力。

Solution

玄学物理题
乱搞发现答案和移动方案无关，且行列独立
一个星星的势能为它到左上角格子的距离的平方，使用一次魔法释放的魔力为在两个星星势能和改变量2\frac{两个星星势能和改变量}{2}2两个星星势能和改变量​。计算初末状态星星势能改变量之和2\frac{初末状态星星势能改变量之和}{2}2初末状态星星势能改变量之和​即可。
