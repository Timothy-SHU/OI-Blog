BZOJ4148【AMPPZ2014】Pillars <构造>

Posted on 2017-11-29 Symbols count in article: 594 Reading time ≈ 2 mins.

Problem

【AMPPZ2014】Pillars

Time Limit: 5Sec5 Sec5Sec
Memory Limit: 256MB256 MB256MB
Special  JudgeSpecial\;JudgeSpecialJudge

Description

给定一个n×mn\times mn×m的矩形，其中有f个2×22\times 22×2的障碍物，其中任意两个障碍物中心之间的欧几里得距离至少为666，
且每个障碍物的中心到边缘的距离至少为333。请找到一条从左下角(1,1)(1,1)(1,1)出发经过所有没有障碍物的点各一次的且最后回到左下角的回路。

Input

第一行包含三个整数n,m,fn,m,fn,m,f(1≤n,m≤10001\le n,m\le 10001≤n,m≤1000且n,mn,mn,m都为偶数)。
接下来fff行，每行两个整数x,yx,yx,y(1≤x<n1\le x<n1≤x<n, 1≤y<m1\le y<m1≤y<m)，表示该障碍物左下角的坐标。

Output

如果无解，输出NIENIENIE，否则第一行输出TAKTAKTAK，第二行输出方案。
方案包含n×m−4×fn\times m-4\times fn×m−4×f个字符，第iii个字符表示第iii步的移动方向，用GGG表示上，DDD表示下，LLL表示左，PPP表示右。

Solution

一道很适合构造入门的题。
我看了ClarisClarisClaris的题解才想出来。
首先不考虑障碍，构造出一个走过全部格子的走法。由于nnn和mmm均为偶数，一定有解。
然后考虑障碍，对障碍四周的格子的方向进行修改。由于两个障碍间距离至少为333，故任两个障碍不会影响，只会有两大类。
