BZOJ3262 陌上花开 < CDQ分治 >

Posted on 2018-05-21 Symbols count in article: 755 Reading time ≈ 3 mins.

Problem

陌上花开

Time  Limit:  20  Sec\mathrm{Time\;Limit:\;20\;Sec}TimeLimit:20Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

有nnn朵花，每朵花有三个属性：花形(sss)、颜色(ccc)、气味(mmm)，用三个整数表示。
现在要对每朵花评级，一朵花的级别是它拥有的美丽能超过的花的数量。
定义一朵花aaa比另一朵花bbb要美丽，当且仅sa≥sb,ca≥cb,ma≥mbs_a\ge s_b,c_a\ge c_b,m_a\ge m_bsa​≥sb​,ca​≥cb​,ma​≥mb​。
显然，两朵花可能有同样的属性。需要统计出评出每个等级的花的数量。

Input

第一行为N,K  (1≤N≤105,1≤K≤2×105)N,K\;(1\le N\le10^5,1\le K\le2\times10^5)N,K(1≤N≤105,1≤K≤2×105)，分别表示花的数量和最大属性值。
以下NNN行，每行三个整数si,ci,mi  (1≤si,ci,mi≤K)s_i,c_i,m_i\;(1\le s_i,c_i,m_i\le K)si​,ci​,mi​(1≤si​,ci​,mi​≤K)，表示第iii朵花的属性。

Output

包含NNN行，分别表示评级为0∼N−10\sim N-10∼N−1的每级花的数量。

Sample Input

Sample Output

标签：CDQ分治 树状数组 三维偏序

Solution

三维偏序，CDQ\mathrm{CDQ}CDQ分治裸题。

首先对xxx排序，记录每朵花排名。分治计算，对于区间[s,t][s,t][s,t]，先按yyy值排序，再考虑该区间[s,mid][s,mid][s,mid]中的花对[mid+1,t][mid+1,t][mid+1,t]中的花产生的贡献，即若排名小于等于midmidmid，将其zzz值加入树状数组；否则统计[s,mid][s,mid][s,mid]中yyy值比它小的花对它的贡献，即为树状数组中[1,z][1,z][1,z]的前缀和。
注意若有相同的一段花，则其答案一定是排在最后的这种花的答案。
