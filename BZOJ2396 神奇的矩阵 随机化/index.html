BZOJ2396 神奇的矩阵 <随机化>

Posted on 2017-11-28 Symbols count in article: 561 Reading time ≈ 2 mins.

Problem

神奇的矩阵

Time Limit: 5Sec5 Sec5Sec
Memory Limit: 512MB512 MB512MB

Description

给出三个行数和列数均为NNN的矩阵AAA、BBB、CCC，判断A×B=CA\times B=CA×B=C是否成立。

Input

题目可能包含若干组数据。
对于每组数据，第一行一个数NNN，接下来给出三个N×NN\times NN×N的矩阵，依次为AAA、BBB、CCC三个矩阵。

Output

对于每组数据，若A×B=CA\times B=CA×B=C成立，则输出YesYesYes，否则NoNoNo。每个答案占一行。

Sample Input

Sample Output

HINT

对于90%90\%90%的数据，NNN不超过100100100；
对于100%100\%100%的数据，NNN不超过100010001000，矩阵中的数字大于等于000小于100010001000，数据组数不超过555组。

标签：矩阵乘法 随机化

Solution

挺巧妙的一道随机化题。
首先考虑直接乘，由于N≤1000N\le 1000N≤1000，而矩乘是O(n3)O(n^3)O(n3)，肯定会TLETLETLE。
我们发现矩阵太大了，那么我们就要压缩矩阵，把这样一个正方形的矩阵压缩为一个行矩阵或列矩阵。
我们先随机出一个1×n1\times n1×n的列矩阵RRR，这样A×B×R=A×(B×R)A\times B\times R=A\times (B\times R)A×B×R=A×(B×R)，将结果与C×RC\times RC×R比较。这样乘法的复杂度就降为O(n2)O(n^2)O(n2)，比较复杂度为O(n)O(n)O(n)，总复杂度为O(n2)O(n^2)O(n2)。如果觉得随机一次不保险，还可以多随机几次，不过只随一次也很容易过。
