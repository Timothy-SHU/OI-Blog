Problem

【ZJOI2010】count 数字计数

Time Limit: 3Sec3 Sec3Sec
Memory Limit: 64MB64 MB64MB

Description

给定两个正整数aaa和bbb，求在[a,b][a,b][a,b]中的所有整数中，每个数码(digit)(digit)(digit)各出现了多少次。

Input

输入文件中仅包含一行两个整数aaa、bbb，含义如上所述。

Output

输出文件中包含一行101010个整数，分别表示0∼90\sim 90∼9在[a,b][a,b][a,b]中出现了多少次。

Sample Input

Sample Output

HINT

30%30\%30%的数据中，a≤b≤106a\le b\le 10^6a≤b≤106
100%100\%100%的数据中，a≤b≤1012a\le b\le 10^{12}a≤b≤1012

标签：数位DP

Solution

BZOJBZOJBZOJ上最简单的数位DPDPDP，比WindyWindyWindy数还水。
预处理数组fff，其中f[i][j]f[i][j]f[i][j]表示第i位为jjj的数共多少个。
对于每次查询，找[1,a][1, a][1,a]和[1,b][1,b][1,b]中0∼90\sim 90∼9出现的次数，按位在fff数组上查，相减即可。
可以用结构体重载运算符，这样一个算式不用写十遍。
