Problem

【USACO2008 Open】奶牛的邻居

Time  Limit:  5  Sec\mathrm{Time\;Limit:\;5\;Sec}TimeLimit:5Sec
Memory  Limit:  64  MB\mathrm{Memory\;Limit:\;64\;MB}MemoryLimit:64MB

Description

了解奶牛们的人都知道，奶牛喜欢成群结队．观察约翰的N  (1≤N≤105)N\;(1\le N\le10^5)N(1≤N≤105)只奶牛，你会发现她们已经结成了几个“群”．每只奶牛在吃草的时候有一个独一无二的位置坐标xi,yi  (xi,yi∈[1,109],xi,yi∈Z)x_i,y_i\;(x_i,y_i\in[1,10^9],x_i,y_i\in\Z)xi​,yi​(xi​,yi​∈[1,109],xi​,yi​∈Z)．当满足下列两个条件之一，两只奶牛iii和jjj是属于同一个群的：

  1. 两只奶牛的曼哈顿距离不超过C  (1≤C≤109)C\;(1\le C\le10^9)C(1≤C≤109)，即∣xi−xj∣+∣yi−yj∣≤C|x_i-x_j|+|y_i-y_j|\le C∣xi​−xj​∣+∣yi​−yj​∣≤C.
  2. 两只奶牛有共同的邻居，即存在一只奶牛kkk，使iii与k，jk，jk，j与kkk均同属一个群．

给出奶牛们的位置，请计算草原上有多少个牛群，以及最大的牛群里有多少奶牛。

Input

第111行输入NNN和CCC，之后NNN行每行输入一只奶牛的坐标。

Output

仅一行，先输出牛群数，再输出最大牛群里的牛数，用空格隔开。

Solution

对于原来的点(x,y)(x,y)(x,y)，将其转化为(x+y,x−y)(x+y,x-y)(x+y,x−y)，则其原来的曼哈顿距离转化为切比雪夫距离，即max⁡(∣xi−xj∣,∣yi−yj∣)\max(|x_i-x_j|,|y_i-y_j|)max(∣xi​−xj​∣,∣yi​−yj​∣)。这样xxx和yyy就无关了。两个点能相连当且仅当转换坐标后∣xi−xj∣≤C,∣yi−yj∣≤C|x_i-x_j|\le C,|y_i-y_j|\le C∣xi​−xj​∣≤C,∣yi​−yj​∣≤C。
将所有点转换坐标后以xxx从小到大排序，这样可以用双指针维护xxx值之差不超过CCC的一段区间。同时对在这段区间中的点用一个set\mathrm{set}set维护其yyy坐标，若新加入的点与其前驱后继的差不超过CCC，则分别连边。用并查集维护连通块个数和大小即可。
