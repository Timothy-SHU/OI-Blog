Problem

【AMPPZ2014】The Cave

Time Limit: 5Sec5 Sec5Sec
Memory Limit: 256MB256 MB256MB
SpecialJudgeSpecial JudgeSpecialJudge

Description

给定一棵有nnn个节点的树，相邻两点之间的距离为111。
请找到一个点xxx，使其满足所有mmm条限制 a b da\ b\ da b d，其中第iii条限制为distx,a+distx,b≤ddist_{x,a}+dist_{x,b}\le ddistx,a​+distx,b​≤d

Input

第一行包含一个正整数ttt(1≤t≤10001\le t\le 10001≤t≤1000)，表示数据组数。
对于每组数据，第一行包含两个正整数n,mn,mn,m(2≤n,m≤3×1052\le n,m\le 3\times 10^52≤n,m≤3×105)，表示点数、限制数。
接下来n−1n-1n−1行，每行两个正整数x,yx,yx,y(1≤x,y≤n1\le x,y\le n1≤x,y≤n)，表示树上的一条边。
接下来mmm行，每行三个正整数a[i],b[i],d[i]a[i],b[i],d[i]a[i],b[i],d[i](1≤a[i],b[i]≤n1\le a[i],b[i]\le n1≤a[i],b[i]≤n, 1≤d[i]≤6×1051\le d[i]\le 6\times 10^51≤d[i]≤6×105)，描述一条限制。
输入数据保证所有nnn之和不超过 3×1053\times 10^53×105，所有mmm之和也不超过 3×1053\times 10^53×105。

Output

输出ttt行。第iii行输出第iii组数据的答案，如果无解输出NIENIENIE，否则输出TAKTAKTAK，
然后输出xxx，如有多组解，输出任意一组。

Solution

好题，完全没想到正解。
设111号节点为根节点。
对于每个限制，所求点离ai→bia_i\to b_iai​→bi​这条路径的距离一定不大于某个数，在根节点确定的情况下，第iii个限制可转化成所求点到根的距离最大为max(0,maxi=1ndep[ai]+dep[bi]−di2)max(0,max_{i=1}^{n}{\frac{dep[a_i]+dep[b_i]−d_i}{2}})max(0,maxi=1n​2dep[ai​]+dep[bi​]−di​​)以内，其中depdepdep表示该点到根节点的距离。
这就可以转化成类似一堆已知某个端点的线段求某个交点的问题。
于是先取限制距离最大的线段上的点，即离根节点最远的点，然后一路往上跑直到某个点刚好满足限制，对于这个点再判断一下是否满足所有情况即可，总复杂度O(n+m)O(n+m)O(n+m)。
