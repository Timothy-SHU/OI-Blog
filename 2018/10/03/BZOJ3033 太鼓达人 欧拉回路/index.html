BZOJ3033 太鼓达人 <欧拉回路>

Posted on 2018-10-03

Problem

太鼓达人

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

七夕祭上，Vani\mathrm{Vani}Vani牵着cl\mathrm{cl}cl的手，在明亮的灯光和欢乐的气氛中愉快地穿行。这时，在前面忽然出现了一台太鼓达人机台，而在机台前坐着的是刚刚被精英队伍成员XLk\mathrm{XLk}XLk、Poet\mathrm{Poet}Poet_shy\mathrm{shy}shy和lydrainbowcat\mathrm{lydrainbowcat}lydrainbowcat拯救出来的的applepi\mathrm{applepi}applepi。看到两人对太鼓达人产生了兴趣，applepi\mathrm{applepi}applepi果断闪人，于是cl\mathrm{cl}cl拿起鼓棒准备挑战。然而即使是在普通难度下，cl\mathrm{cl}cl的路人本性也充分地暴露了出来。一曲终了，不但没有过关，就连鼓都不灵了。Vani\mathrm{Vani}Vani十分过意不去，决定帮助工作人员修鼓。
鼓的主要元件是MMM个围成一圈的传感器。每个传感器都有开和关两种工作状态，分别用111和000表示。显然，从不同的位置出发沿顺时针方向连续检查KKK个传感器可以得到MMM个长度为KKK的010101串。Vani\mathrm{Vani}Vani知道这MMM个010101串应该是互不相同的。而且鼓的设计很精密，MMM会取到可能的最大值。现在Vani\mathrm{Vani}Vani已经了解到了K的值，他希望你求出MMM的值，并给出字典序最小的传感器排布方案。

Input

一个整数KKK。

Output

一个整数MMM和一个二进制串，由一个空格分隔。表示可能的最大的MMM以及字典序最小的排布方案，字符000表示关，111表示开。你输出的串的第一个字和最后一个字是相邻的。

Hint

Solution

将k−1k-1k−1位二进制数看成点，kkk位二进制数看成边，连接u,vu,vu,v两点的边权就是(u<<1)∣v(u<<1)|v(u<<1)∣v。
例如k=4k=4k=4时，边010→101010\to 101010→101的权为010101010101，整个图如下：

![](https://i.loli.net/2018/10/08/5bbaff6e70a6c.png)
*（转载自 Clove_unique 的博客）* 显然每个点的出度和入度均为$2$，一定存在欧拉回路，而欧拉回路一定会经过所有的边，即凑出所有$k$位二进制数。于是第一问答案为$2^k$，第二问在图上找出字典序最小的欧拉回路即可。
