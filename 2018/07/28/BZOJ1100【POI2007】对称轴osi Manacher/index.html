BZOJ1100【POI2007】对称轴osi < Manacher >

Posted on 2018-07-28

Problem

【POI2007】对称轴osi

$\mathrm{Time;Limit:;10;Sec}$
$\mathrm{Memory;Limit:;162;MB}$

Description

$\mathrm{FGD}$小朋友——一个闻名遐迩的年轻数学家——有一个小MM，$\mathrm{yours}$。$\mathrm{FGD}$小朋友非常喜欢他的MM，所以他很乐意帮助他的MM做数学作业。但是，就像所有科学的容器一样，FGD的大脑拒绝不停地重复思考同样的问题。不幸的是，$\mathrm{yours}$是一个十分用功的学生，所以她不停地让$\mathrm{FGD}$帮助她检查她的作业。
一个阳光明媚的周末，$\mathrm{yours}$的数学老师布置了非常多的寻找多边形的对称轴的题，足够她做相当长的一段时间了。在此之前$\mathrm{FGD}$已经决定去海边度过这个难得的假期，不过他还是觉得应该帮助他的MM对付可爱的数学作业。很快地，他找到了解决方案，最好写一个程序来帮助$\mathrm{yours}$检查她的数学作业。
因为$\mathrm{FGD}$并非一个计算机科学家，所以他找到了他的好朋友你，请你帮助他完成这个任务。
请写一个程序：读入多边形的描述计算出每个多边形的对称轴数将计算的结果输出。

Input

输入的第一行包含一个正整数$T;(1\le T\le10)$，为多边形的边数。
接下来，为$T$个多边形的描述。
每个描述的第一行为一个正整数$n;(3\le n\le10^5)$，表示了多边形的点数。
后面$n$行每行两个整数$x$和$y;(-10^8\le x, y\le10^8)$，依次表示多边形的顶点坐标。
多边形不一定是凸的，但是不自交。任何两条边都只有最多一个公共点，即它们的公共端点。此外，没有两条连续的边平行。

Output

你的程序应该输出正好$T$行，第k行包含了一个整数$n_k$——表示第$k$个多边形有多少个对称轴。

Sample Input

Sample Output

标签：Manacher

Solution

$\mathrm{Manacher}$好题。

如果我们确定一个多边形的每条边和每个角，我们就确定了这个多边形。那么半个多边形呢？也是一样的。
我们将$n$边形写成长为$2n$的数组，即每条边的长度和两边夹角的叉积交错排列。将这个数组看成一个环，对称意味着这个环关于某个位置对称。按常理断环成链后跑$\mathrm{Manacher}$，过程中判每个位置的回文半径是否大于$n$即可。
