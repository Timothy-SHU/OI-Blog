BZOJ3157 国王奇遇记 <倍增>

Posted on 2018-09-19

Problem

国王奇遇记

$\mathrm{Time;Limit:;10;Sec}$
$\mathrm{Memory;Limit:;512;MB}$

Description

![](https://i.loli.net/2018/09/19/5ba22abcf2704.jpg)
### Input 共一行，包括两个正整数$N$和$M$。 ### Output 共一行，为所求表达式的值对$10^9+7$取模的值。

Sample Input

Sample Output

Hint

$1\le N\le10^9$，$1\le m\le200$

Source

$\mathrm{Katharon}$

标签：倍增

Solution

倍增处理幂和。
令
$$
f(n,k)=\sum_{i=1}^{n}i^km^i
$$
则
$$
\begin{aligned}
f(n+1,k)
&=\sum_{i=1}^{n+1}i^km^i\
&=m+\sum_{i=2}^{n+1}i^km^i\
&=m+\sum_{i=1}^{n}(i+1)^km^{i+1}\
&=m+m\sum_{i=1}^{n}(i+1)^km^i\
&=m+m\sum_{i=1}^{n}m^i\sum_{j=0}^{k}\binom{k}{j}i^j\
&=m+m\sum_{j=0}^{k}\binom{k}{j}\sum_{i=1}^{n}i^jm^i\
&=m+m\sum_{j=0}^{k}\binom{k}{j}f(n,j)\
f(2n,k)
&=\sum_{i=1}^{2n}i^km^i\
&=\sum_{i=1}^{n}i^km^i+\sum_{i=n+1}^{2n}i^km^i\
&=f(n,k)+\sum_{i=1}^{n}(i+n)^km^{i+n}\
&=f(n,k)+m^n\sum_{i=1}^{n}(i+n)^km^i\
&=f(n,k)+m^n\sum_{i=1}^{n}m^i\sum_{j=0}^{k}\binom{k}{j}i^jn^{k-j}\
&=f(n,k)+m^n\sum_{j=0}^{k}\binom{k}{j}n^{k-j}\sum_{i=1}^{n}i^jm^i\
&=f(n,k)+m^n\sum_{j=0}^{k}\binom{k}{j}n^{k-j}f(n,j)\
\end{aligned}
$$
于是可以将规模为$n$的问题根据奇偶性降低到规模为$n-1$和$\frac{n}{2}$的问题，递归处理即可。
