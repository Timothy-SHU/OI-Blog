BZOJ1415【NOI2005】聪聪和可可 <概率DP+最短路>

Posted on 2018-09-27 Edited on 2018-09-29

Problem

【NOI2005】聪聪和可可

$\mathrm{Time;Limit:;10;Sec}$
$\mathrm{Memory;Limit:;162;MB}$

Description

![](https://www.lydsy.com/JudgeOnline/images/1415_1.jpg)
### Input 第一行为两个整数$N$和$E$，以空格分隔，分别表示森林中的景点数和连接相邻景点的路的条数。 第二行包含两个整数$C$和$M$，以空格分隔，分别表示初始时聪聪和可可所在的景点的编号。 接下来$E$行，每行两个整数，第$i+2$行的两个整数$A_i$和$B_i$表示景点$A_i$和景点$B_i$之间有一条无向边。 输入保证任何两个景点之间不会有多于一条路直接相连，且聪聪和可可之间必有路直接或间接的相连。 ### Output 输出一个实数，四舍五入保留三位小数，表示平均多少个时间单位后聪聪会把可可吃掉。 ### Sample Input **Input #1** **Input #2** ### Sample Output **Output #1** **Output #2** ### HINT 对于$50\%$的数据，$1\le N\le50$。 对于所有的数据，$1\le N,E\le1000$。

标签：概率DP 最短路

Solution

经典概率$\mathrm{DP}$。

暴力$\mathrm{BFS}$预处理数组$nxt[][]$，其中$nxt[x][y]$表示聪聪在$x$，可可在$y$时，聪聪下一步会走到哪个点。
用$f[c][k]$表示聪聪在$c$，可可在$k$时，期望下还需多少单位时间可可才会被抓住。显然状态之间是不会存在环的，这是因为两者间的距离会不断缩小。
状态转移：

  • 若$c=k$，则$f[c][k]=0$
  • 若$dis(c,k)\le2$，则$f[c][k]=1$
  • 其他情况下，令$p=nxt[nxt[c][k]][k]$，则有
    $$
    f[c][k]=\frac{f[p][k]+\sum_{k\to q}f[p][q]}{deg(k)+1}+1
    $$

在记忆化搜索时特判一下即可。
