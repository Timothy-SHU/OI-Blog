BZOJ3143【HNOI2013】游走 <高斯消元>

Posted on 2018-05-04

Problem

【HNOI2013】游走

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

一个无向连通图，顶点从111编号到NNN，边从111编号到MMM。
小Z\mathrm{小Z}小Z在该图上进行随机游走，初始时小Z\mathrm{小Z}小Z在111号顶点，每一步小Z\mathrm{小Z}小Z以相等的概率随机选择当前顶点的某条边，沿着这条边走到下一个顶点，获得等于这条边的编号的分数。当小Z\mathrm{小Z}小Z到达NNN号顶点时游走结束，总分为所有获得的分数之和。
现在请你对这MMM条边进行编号，使得小Z\mathrm{小Z}小Z获得的总分的期望值最小。

Input

第一行是正整数NNN和MMM，分别表示该图的顶点数和边数。
接下来MMM行每行是整数u,v  (1≤u,v≤N)u,v\;(1\le u,v\le N)u,v(1≤u,v≤N)，表示顶点uuu与顶点vvv之间存在一条边。

Output

仅包含一个实数，表示最小的期望值，保留333位小数。

Sample Output

Explanation

边(1,2)(1,2)(1,2)编号为111，边(1,3)(1,3)(1,3)编号222，边(2,3)(2,3)(2,3)编号为333。

HINT

2≤N≤5002\le N\le5002≤N≤500
保证图为无向简单连通图

标签：高斯消元

Solution

大致思路是求出每条边的期望经过次数，然后贪心选择边权。
然而对于边来说，并不好确定其期望经过次数，而点的期望经过次数则更好求。

设xix_ixi​为到达点iii并继续向下一个点移动的期望次数。那么对于普通的点（非起点111或终点nnn），其只能从与其相邻的点走过来，于是对于点iii，从iii连出去的点的集合为SiS_iSi​，那么xi=∑j∈Sixjdjx_i=\sum_{j\in S_i}\frac{x_j}{d_j}xi​=∑j∈Si​​dj​xj​​，其中ddd为点的度数。而对于111号点，初始时一定在111号点上，因此一定初始就有111次的经过次数，而游走过程中的情况与普通点相同，于是x1=∑j∈S1xjdjx_1=\sum_{j\in S_1}\frac{x_j}{d_j}x1​=∑j∈S1​​dj​xj​​。对于nnn号点，由于到了nnn后不能继续走动，即只进不出，于是一定不会经过（因为前面经过的定义是要有向下一个点移动的可能），因此xn=0x_n=0xn​=0。
于是有方程组

对于一条边iii，其端点为uuu和vvv，要经过iii必定只能是从uuu到vvv或从vvv到uuu。在uuu时有1du\frac{1}{d_u}du​1​的概率走这条边，在vvv时有1dv\frac{1}{d_v}dv​1​的概率走这条边，于是第iii条边的期望经过次数为yi=xudu+xvdvy_i=\frac{x_u}{d_u}+\frac{x_v}{d_v}yi​=du​xu​​+dv​xv​​。
