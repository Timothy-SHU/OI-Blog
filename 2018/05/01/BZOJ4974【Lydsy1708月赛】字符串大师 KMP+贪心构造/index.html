Problem

【Lydsy1708月赛】字符串大师

Time  Limit:  1  Sec\mathrm{Time\;Limit:\;1\;Sec}TimeLimit:1Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

一个串TTT是SSS的循环节，当且仅当存在正整数kkk，使得SSS是TTT重复kkk次的前缀，比如abcd是abcdabcdab的循环节。
给定一个长度为nnn的仅由小写字符构成的字符串SSS，请对于每个k  (1≤k≤n)k\;(1\le k\le n)k(1≤k≤n)，求出SSS长度为kkk的前缀的最短循环节的长度periper_iperi​。
字符串大师小Q\mathrm{小Q}小Q觉得这个问题过于简单，于是花了一分钟将其AC\mathrm{AC}AC了，他想检验你是否也是字符串大师。
小Q\mathrm{小Q}小Q告诉你nnn以及per1,per2,⋯ ,pernper_1,per_2,\cdots,per_nper1​,per2​,⋯,pern​，请找到一个长度为nnn的小写字符串SSS，使得SSS能对应上perperper。

Input

第一行包含一个正整数n  (1≤n≤105)n\;(1\le n\le10^5)n(1≤n≤105)，表示字符串的长度。
第二行包含nnn个正整数per1,per2,⋯ ,pern  (1≤peri≤i)per_1,per_2,\cdots,per_n\;(1\le per_i\le i)per1​,per2​,⋯,pern​(1≤peri​≤i)，表示每个前缀的最短循环节长度。
输入数据保证至少存在一组可行解。

Output

输出一行一个长度为nnn的小写字符串SSS，即某个满足条件的SSS。
若有多个可行的SSS，输出字典序最小的那一个。

Solution

★\bigstar★本文中所有数组和字符串下标从000开始。

首先有一个结论：nxti=i−perinxt_i=i-per_inxti​=i−peri​
证明：
对于字符串PPP，其最短循环节为RRR，除去循环节后多余的部分为QQQ，如图所示。

![1.PNG](https://i.loli.net/2018/05/02/5ae96516e82ca.png)

那么再在上面接一个RRR，一定可以包含PPP，于是可以知道QQQ一定是RRR的前缀，所以有下图：

![2.PNG](https://i.loli.net/2018/05/02/5ae96516e8d6d.png)

将PPP末尾循环节长度那么长去掉，得到P1P_1P1​，将PPP第一个循环节去掉，得到P2P_2P2​，发现两者是相同的（如下图）。而这显然是PPP的boarderboarderboarder，所以P1P_1P1​的末尾位置为n−pern-pern−per，即nxt[n]=n−pernxt[n]=n-pernxt[n]=n−per

![3.PNG](https://i.loli.net/2018/05/02/5ae965170167b.png)

这样根据给出的perperper可以将nxtnxtnxt数组处理出来。
从前往后构造，对于位置iii：

  • nxti≠−1nxt_i\ne-1nxti​=−1，一定有s[i]=s[nxti]s[i]=s[nxt_i]s[i]=s[nxti​]，可以直接赋值
  • nxti=−1nxt_i=-1nxti​=−1，那么在计算nxtnxtnxt的过程中，即将这个串与自己做匹配的时候，不断根据nxtnxtnxt向前跳到的位置一定不会和当前位置匹配，否则nxti=最先能匹配的位置nxt_i=最先能匹配的位置nxti​=最先能匹配的位置。于是将能向前跳到的位置上的字符存下来，找一个最小的没有跳到过的字符作为这一位置的字符

如此贪心构造即可得到最优解。
