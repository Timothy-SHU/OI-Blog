Description

Let’s denote d(n)d(n)d(n) as the number of divisors of a positive integer nnn. You are given three integers aaa, bbb and ccc. Your task is to calculate the following sum:
∑i=1a∑j=1b∑k=1cd(i⋅j⋅k)\sum_{i=1}^{a}\sum_{j=1}^{b}\sum_{k=1}^{c}d(i·j·k)∑i=1a​∑j=1b​∑k=1c​d(i⋅j⋅k)
Find the sum modulo 1073741824(230)1073741824 (2^{30})1073741824(230).

Input

The first line contains three space-separated integers aaa, bbb and ccc (1≤a,b,c≤2000)(1\le a,b,c\le 2000)(1≤a,b,c≤2000).

Output

Print a single integer — the required sum modulo 1073741824(230)1073741824 (2^{30})1073741824(230).
