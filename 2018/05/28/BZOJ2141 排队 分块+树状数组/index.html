BZOJ2141 排队 <分块+树状数组>

Posted on 2018-05-28

Problem

排队

Time  Limit:  4  Sec\mathrm{Time\;Limit:\;4\;Sec}TimeLimit:4Sec
Memory  Limit:  259  MB\mathrm{Memory\;Limit:\;259\;MB}MemoryLimit:259MB

Description

排排坐，吃果果，生果甜嗦嗦，大家笑呵呵。你一个，我一个，大的分给你，小的留给我，吃完果果唱支歌，大家乐和和。
红星幼儿园的小朋友们排起了长长地队伍，准备吃果果。不过因为小朋友们的身高有所区别，排成的队伍高低错乱，极不美观。
设第iii个小朋友的身高为hih_ihi​，我们定义一个序列的杂乱程度为满足i>ji>ji>j且hi<hjh_i<h_jhi​<hj​的(i,j)(i,j)(i,j)数量。
幼儿园阿姨每次会选出两个小朋友，交换他们的位置，请你帮忙计算出每次交换后，序列的杂乱程度。
为方便幼儿园阿姨统计，在未进行任何交换操作时，你也应该输出该序列的杂乱程度。

Input

第一行为一个正整数nnn，表示小朋友的数量。
第二行包含nnn个由空格分隔的正整数h1,h2,⋯ ,hnh_1,h_2,\cdots,h_nh1​,h2​,⋯,hn​，依次表示初始队列中小朋友的身高。
第三行为一个正整数mmm，表示交换操作的次数。
以下mmm行每行包含两个正整数aia_iai​和bib_ibi​，表示交换位置aia_iai​与位置bib_ibi​的小朋友。

Output

输出文件共mmm行，第iii行一个正整数表示交换操作iii结束后，序列的杂乱程度。

操作111结束后，序列为{130,140,150}\lbrace130,140,150\rbrace{130,140,150}，不存在满足条件的(i,j)(i,j)(i,j)；

1≤m≤2×1031\le m\le2\times10^31≤m≤2×103，1≤n≤2×1041\le n\le2\times10^41≤n≤2×104，1≤hi≤1091\le h_i\le10^91≤hi​≤109，ai≠bia_i\ne b_iai​=bi​，1≤ai,bi≤n1\le a_i,b_i\le n1≤ai​,bi​≤n。

标签：分块 树状数组

Solution

分块基础应用之带交换逆序对。

将原序列分为n\sqrt{n}n​个大小为n\sqrt{n}n​的块，每块内维护一个值域树状数组，记录每个值的个数。
一开始增量计算总逆序对对数，随后对每个操作计算会增加/减少多少对逆序对。
当交换a[x],a[y]  (x<y)a[x],a[y]\;(x<y)a[x],a[y](x<y)时，对于i∈[x,y]i\in[x,y]i∈[x,y]，

  • 若a[i]<a[x]a[i]<a[x]a[i]<a[x]，则ans−−ans- -ans−−
  • 若a[i]>a[x]a[i]>a[x]a[i]>a[x]，则ans++ans++ans++
  • 若a[i]<a[y]a[i]<a[y]a[i]<a[y]，则ans++ans++ans++
  • 若a[i]>a[y]a[i]>a[y]a[i]>a[y]，则ans−−ans- -ans−−

整块直接用树状数组统计，剩余部分暴力即可。
