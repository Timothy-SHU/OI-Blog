BZOJ5210 最大连通子块和 <树链剖分+树形DP>

Posted on 2018-06-05 Edited on 2018-06-10

Problem

最大连通子块和

$\mathrm{Time;Limit:;20;Sec}$
$\mathrm{Memory;Limit:;128;MB}$

Description

给出一棵$n$个点，以$1$为根的有根树，点有点权。
要求支持如下两种操作：

  1. $\mathrm{M};x;y$：将点$x$的点权改为$y$
  2. $\mathrm{Q};x$：求以$x$为根的子树的最大连通子块和

一棵子树的最大连通子块和指该子树所有子连通块的点权和中的最大值（本题中子连通块包括空连通块，点权和为$0$）。

Input

第一行两个整数$n$,$m$，表示树的点数以及操作的数目。
第二行$n$个整数，第$i$个整数$w_i$表示第$i$个点的点权。
接下来的$n-1$行，每行两个整数$x,y$，表示$x$和$y$之间有一条边相连。
接下来的$m$行，每行输入一个操作，含义如题目所述。
保证操作为$\mathrm{M};x;y$或$\mathrm{Q};x$之一。

Output

对于每个$\mathrm{Q}$操作输出一行一个整数，表示询问子树的最大连通子块和。

HINT

$1\le n,m\le2\times10^5$，任意时刻$|w_i|\le10^9$ 。

Source

标签：树链剖分 树上DP

Solution

经典树链剖分维护树上$\mathrm{DP}$。以下解法源自出题人CQzhangyu的博客和GXZlegend的博客。

首先考虑暴力$\mathrm{DP}$，令$f[x]$表示$x$子树中包含$x$的连通块权值和最大值，那么$f[x]=\max(0, w[x]+\sum_{x\to y\in\mathrm{G}}f[y])$。维护$s[x]$表示$x$子树中连通块权值和最大值，则$s[x]=\max(f[x],\max_{x\to y\in\mathrm{G}}s[y])$。每次修改后重新$\mathrm{DP}$，可做到$O(n^2)$。

注意到每次修改后不是所有的$f$都变化。用树链剖分维护树上$\mathrm{DP}$，可以每次不修改所有的$f$值。然而直接维护$f$的值不方便，因为递推式中的和式在线段树上不便于计算。于是引入$g[x]=w[x]+\sum f[y]$，其中$y$是$x$的轻儿子。那么重链上的转移就变为$f[x]=\max(0,f[son[x]]+g[x])$。这其实就是最大连续子段和的$\mathrm{DP}$方式，可以用线段树维护带修改最大连续子段和。于是对于每次修改，向上跳重链，在线段树上每条重链的区域内维护最大连续子段和即可。注意这里“每条重链的区域”指的是链顶到链底的距离，而非括号序列。

再考虑如何维护$s$。注意到$s$每次都直接取最值，带修改后，其实是可删除堆的形式。对每个点维护可删除堆来维护轻儿子的$s$值最大值，对于重儿子则在线段树上维护。在线段树上修改后$\mathrm{update}$时，用每个结点的$s$堆顶元素更新最大子段和，即可动态维护$s$。

查询时，直接向上跳重链，将该重链对应区间的最大子段和取出来打擂即可。

这样修改复杂度$O(\log^2n)$，查询复杂度$O(\log n)$，总复杂度$O(m\log^2n)$。
