BZOJ4025 二分图 <线段树分治+并查集>

Posted on 2018-06-29 Symbols count in article: 838 Reading time ≈ 3 mins.

Problem

二分图

Time  Limit:  20  Sec\mathrm{Time\;Limit:\;20\;Sec}TimeLimit:20Sec
Memory  Limit:  512  MB\mathrm{Memory\;Limit:\;512\;MB}MemoryLimit:512MB

Description

神犇有一个nnn个节点的图。
因为神犇是神犇，所以在T时间内一些边会出现后消失。
神犇要求出每一时间段内这个图是否是二分图。
这么简单的问题神犇当然会做了，于是他想考考你。

Input

输入数据的第一行是三个整数n,m,Tn,m,Tn,m,T。
第222行到第m+1m+1m+1行，每行4个整数u,v,start,endu,v,start,endu,v,start,end，表示第iii条边连接u,vu,vu,v两个点，这条边在startstartstart时刻出现，在第endendend时刻消失。

Output

输出包含TTT行。在第iii行中，如果第iii时间段内这个图是二分图，那么输出Yes，否则输出No。

Sample Input

Sample Output

Explanation

000时刻，出现两条边1−21-21−2和2−32-32−3。
第111时间段内，这个图是二分图，输出YesYesYes。
111时刻，出现一条边1−31-31−3。
第222时间段内，这个图不是二分图，输出NoNoNo。
222时刻，1−21-21−2和1−31-31−3两条边消失。
第333时间段内，只有一条边2−32-32−3，这个图是二分图，输出YesYesYes。

HINT

n≤105n\le10^5n≤105，m≤2×105m\le2\times10^5m≤2×105，T≤105T\le10^5T≤105，1≤u,v≤n1\le u,v\le n1≤u,v≤n，0≤start≤end≤T0\le start\le end\le T0≤start≤end≤T。

标签：线段树分治 并查集

Solution

事实证明，线段树不仅是一种数据结构，更是一种思想。

先考虑不带存在时间的版本，可以直接用带权并查集维护。

我们将整个时间想成一段线段，那么其中的任意时间段都是一条更短的线段，显然是可以和线段树一样分作log⁡n\log{n}logn段。那么我们对每条边存在的时间区间做这种拆分，在这log⁡n\log{n}logn个区间内的并查集中分别加入这条边然后判断询问。

像整体二分一样分治处理即可。
