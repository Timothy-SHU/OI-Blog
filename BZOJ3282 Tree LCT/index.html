BZOJ3282 Tree < LCT >

Posted on 2018-01-15 Symbols count in article: 809 Reading time ≈ 3 mins.

Problem

Tree

Time Limit: 30Sec30 Sec30Sec
Memory Limit: 512MB512 MB512MB

Description

给定NNN个点以及每个点的权值，要你处理接下来的MMM个操作。操作有444种。操作从000到333编号。点从111到NNN编号。
000.后接两个整数 (x,y)(x, y)(x,y)，代表询问从xxx到yyy的路径上的点的权值的xorxorxor和。保证xxx到yyy是联通的。
111.后接两个整数 (x,y)(x, y)(x,y)，代表连接xxx到yyy，若x到y已经联通则无需连接。
222.后接两个整数 (x,y)(x, y)(x,y)，代表删除边$ (x, y)$，不保证边 (x,y)(x, y)(x,y) 存在。
333.后接两个整数 (x,y)(x, y)(x,y)，代表将点xxx上的权值变成yyy。

Input

第111行两个整数，分别为NNN和MMM，代表点数和操作数。
第222行到第N+1N+1N+1行，每行一个整数，整数在 [1,109][1, 10^9][1,109] 内，代表每个点的权值。
第N+2N+2N+2行到第N+M+1N+M+1N+M+1行，每行三个整数，分别代表操作类型和操作所需的量。

Output

对于每一个000号操作，你须输出xxx到yyy的路径上点权的xorxorxor和。
