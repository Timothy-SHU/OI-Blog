BZOJ1594【USACO2008 Jan】猜数游戏 <二分答案+线段树>

Posted on 2018-09-05 Symbols count in article: 1.1k Reading time ≈ 4 mins.

Problem

【Usaco2008 Jan】猜数游戏

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  162  MB\mathrm{Memory\;Limit:\;162\;MB}MemoryLimit:162MB

Description

为了提高自己低得可怜的智商，奶牛们设计了一个新的猜数游戏，来锻炼她们的逻辑推理能力。
游戏开始前，一头指定的奶牛会在牛棚后面摆N  (1≤N≤106)N\;(1\le N\le 10^6)N(1≤N≤106)堆干草，每堆有若干捆，并且没有哪两堆中的草一样多。
所有草堆排成一条直线，从左到右依次按1∼N1\sim N1∼N编号，每堆中草的捆数在[1,109][1,10^9][1,109]之间。
然后，游戏开始。
另一头参与游戏的奶牛会问那头摆干草的奶牛Q  (1≤Q≤25000)Q\;(1\le Q\le 25000)Q(1≤Q≤25000)个问题，问题的格式如下：
编号为Ql∼Qh  (1≤Ql≤Qh≤N)Q_l\sim Q_h\;(1\le Q_l\le Q_h\le N)Ql​∼Qh​(1≤Ql​≤Qh​≤N)的草堆中，最小的那堆里有多少捆草？
对于每个问题，摆干草的奶牛回答一个数字AAA，但或许是不想让提问的奶牛那么容易地得到答案，又或许是她自己可能记错每堆中干草的捆数，总之，她的回答不保证是正确的。
请你帮助提问的奶牛判断一下，摆干草的奶牛的回答是否有自相矛盾之处。

Input

  • 第111行: 222个用空格隔开的整数：NNN和QQQ
  • 第2∼Q+12\sim Q+12∼Q+1行: 每行为333个用空格隔开的整数Ql,Qh,AQ_l,Q_h,AQl​,Qh​,A，描述了一个问题以及它对应的回答

Output

如果摆干草的奶牛有可能完全正确地回答了这些问题，输出000；否则输出111个1∼Q1\sim Q1∼Q中的数，表示这个问题的答案与它之前的那些回答有冲突之处。
注意：如果有冲突出现输出一个数mmm，使得前m−1m-1m−1个命题不冲突。

Sample Output

Source

标签：二分答案 线段树

Solution

二分答案，设二分到mmm，需要判断第1∼m1\sim m1∼m个条件是否不存在矛盾（最后答案加一）。

将第1∼m1\sim m1∼m个条件按AAA从大到小排序，每次找出一段AAA相同的条件，求其区间的并。可知值为AAA的数一定在这个并中。因此若这个并不存在，则一定矛盾。另一种矛盾的情况是存在一个包含这个并的区间，其答案大于AAA。对于这种情况，我们在每次操作完一组AAA相同的条件后，对这些区间的交做标记。若一组AAA相同的条件的区间并完全被打上标记，意味着其被前面的AAA更大的区间覆盖，即存在一个包含该并的区间，其答案大于AAA。这个标记可以在线段树上维护。

综上，checkcheckcheck的复杂度是O(mlog⁡N)O(m\log{N})O(mlogN)，总时间复杂度为O(Qlog⁡Nlog⁡Q)O(Q\log{N}\log{Q})O(QlogNlogQ)。
