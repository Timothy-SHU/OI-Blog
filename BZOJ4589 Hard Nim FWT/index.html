Problem

Hard Nim

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

Claris\mathrm{Claris}Claris和NanoApe\mathrm{NanoApe}NanoApe在玩石子游戏，他们有nnn堆石子，规则如下：

  1. Claris\mathrm{Claris}Claris和NanoApe\mathrm{NanoApe}NanoApe两个人轮流拿石子，Claris\mathrm{Claris}Claris先拿。
  2. 每次只能从一堆中取若干个，可将一堆全取走，但不可不取，拿到最后1颗石子的人获胜。

不同的初始局面，决定了最终的获胜者，有些局面下先拿的Claris\mathrm{Claris}Claris会赢，其余的局面Claris\mathrm{Claris}Claris会负。
Claris\mathrm{Claris}Claris很好奇，如果这nnn堆石子满足每堆石子的初始数量是不超过mmm的质数，而且他们都会按照最优策略玩游戏，那么NanoApe\mathrm{NanoApe}NanoApe能获胜的局面有多少种。
由于答案可能很大，你只需要给出答案对109+710^9+7109+7取模的值。

Input

输入文件包含多组数据，以EOF为结尾。
对于每组数据，输入一行两个正整数nnn和mmm。

Output

对于每组数据，输出一行一个整数表示答案。
