BZOJ1877【SDOI2009】晨跑 <拆点费用流>

Posted on 2017-12-25 Symbols count in article: 846 Reading time ≈ 3 mins.

Problem

【SDOI2009】晨跑

Time  Limit:  4  Sec\mathrm{Time\;Limit:\;4\;Sec}TimeLimit:4Sec
Memory  Limit:  64  MB\mathrm{Memory\;Limit:\;64\;MB}MemoryLimit:64MB

Description

ElaxiaElaxiaElaxia最近迷恋上了空手道，他为自己设定了一套健身计划，比如俯卧撑、仰卧起坐等 等，不过到目前为止，他坚持下来的只有晨跑。 现在给出一张学校附近的地图，这张地图中包含NNN个十字路口和MMM条街道，ElaxiaElaxiaElaxia只能从 一个十字路口跑向另外一个十字路口，街道之间只在十字路口处相交。ElaxiaElaxiaElaxia每天从寝室出发 跑到学校，保证寝室编号为111，学校编号为NNN。 ElaxiaElaxiaElaxia的晨跑计划是按周期（包含若干天）进行的，由于他不喜欢走重复的路线，所以在一个周期内，每天的晨跑路线都不会相交（在十字路口处），寝室和学校不算十字路 口。ElaxiaElaxiaElaxia耐力不太好，他希望在一个周期内跑的路程尽量短，但是又希望训练周期包含的天数尽量长。 除了练空手道，ElaxiaElaxiaElaxia其他时间都花在了学习和找MMMMMM上面，所有他想请你帮忙为他设计 一套满足他要求的晨跑计划。

Input

第一行：两个数N,MN,MN,M。表示十字路口数和街道数。
接下来MMM行，每行333个数a,b,ca,b,ca,b,c，表示路口aaa和路口bbb之间有条长度为ccc的街道（单向）。

Output

两个数，第一个数为最长周期的天数，第二个数为满足最长天数的条件下最短的路程长度。

Sample Input

Sample Output

Hint

N≤200,M≤20000N\le 200, M\le 20000N≤200,M≤20000。

标签：拆点费用流

Solution

拆点费用流套路题。
首先要同时考虑天数最长和路程最短两个权，可知要用费用流。
看到每个点只经过一次可知要把每个点拆成入点和出点，中间连流量为111费用为000的边。
套路建图后跑费用流即可。
