AGC026E Synchronized Subsequence <贪心+DP>

Posted on 2018-08-01 Symbols count in article: 839 Reading time ≈ 3 mins.

Problem

Synchronized Subsequence

Time  limit:  2  Sec\mathrm{Time\;limit:\;2\;Sec}Timelimit:2Sec
Memory  limit:  1024  MB\mathrm{Memory\;limit:\;1024\;MB}Memorylimit:1024MB

Statement

You are given a string SSS of length 2N2N2N, containing NNN occurrences of a and NNN occurrences of b.
You will choose some of the characters in SSS. Here, for each i=1,2,⋯ ,Ni=1,2,\cdots,Ni=1,2,⋯,N, it is not allowed to choose exactly one of the following two: the ithi^{th}ith occurrence of a and the ithi^{th}ith occurrence of b. (That is, you can only choose both or neither.) Then, you will concatenate the chosen characters (without changing the order).
Find the lexicographically largest string that can be obtained in this way.

Constraints

1≤N≤30001\le N\le30001≤N≤3000
SSS is a string of length 2N2N2N containing NNN occurrences of a and NNN occurrences of b.

Input

Input is given from Standard Input in the following format:
NSN\\SNS

Output

Print the lexicographically largest string that satisfies the condition.

Translation

给出一个长为2N2N2N的仅由a和b组成的字符串，每次可以删除第iii个a和第iii个b。求最终形成的字典序最大的串。

Solution

首先将整个串分为若干段，每一段都保证a的个数和b的个数相等。
设第iii对a,b的位置分别为pi,qip_i,q_ipi​,qi​，那么每一段中满足以下两种条件之一：

  1. ∀i,  pi<qi\forall i,\;p_i<q_i∀i,pi​<qi​
  2. ∀i,  pi>qi\forall i,\;p_i>q_i∀i,pi​>qi​

对于情况1情况1情况1，显然最后的串是abab...的形式，于是直接贪心从前往后选，对于当前这一对(a,b)(a,b)(a,b)如果a的位置在上一次选的(a,b)(a,b)(a,b)的b的位置之后，即可选当前这一对。
对于情况2情况2情况2，如果选了中间的某一组(b,a)(b,a)(b,a)，则其后面的(b,a)(b,a)(b,a)都选一定比不选要优。于是枚举哪一组(b,a)(b,a)(b,a)开始选，贪心求出字典序最大的串。
如此，可以将每一段都处理成能变成的字典序最大的串。最后需要考虑的就是拼起来。
显然，对于某一段，只有选和不选两种可能，于是用f[i]f[i]f[i]表示选第iii段做开头，后面的串可以任意选，能得到的字典序最大的串（这里fff是一个stringstringstring数组）。那么f[i]=max⁡j>i{S[i]+f[j]}f[i]=\max_{j>i}\lbrace S[i]+f[j]\rbracef[i]=maxj>i​{S[i]+f[j]}。这样可以O(n2)O(n^2)O(n2)做DP\mathrm{DP}DP。最后的答案为ans=max⁡i=1m{f[i]}ans=\max_{i=1}^{m}\lbrace f[i]\rbraceans=maxi=1m​{f[i]}。
