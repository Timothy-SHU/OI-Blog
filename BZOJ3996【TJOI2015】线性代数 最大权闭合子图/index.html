BZOJ3996【TJOI2015】线性代数 <最大权闭合子图>

Posted on 2018-06-25 Symbols count in article: 996 Reading time ≈ 4 mins.

Problem

【TJOI2015】线性代数

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

给出一个N×NN\times NN×N的矩阵BBB和一个1×N1\times N1×N的矩阵CCC。
求出一个1×N1\times N1×N的010101矩阵AAA，使得D=(A×B−C)×ATD=(A\times B-C)\times A^TD=(A×B−C)×AT最大，输出最大的DDD值。

Input

第一行输入一个整数NNN。
接下来NNN行输入BBB矩阵，第iii行第jjj个数字代表Bi,jB_{i,j}Bi,j​.
接下来一行输入NNN个整数，代表矩阵CCC。
矩阵BBB和矩阵CCC中每个数字都是不超过100010001000的非负整数。

Output

输出最大的DDD。

Sample Input

Sample Output

Hint

1≤N≤5001\le N\le5001≤N≤500

标签：网络流 最大权闭合子图

Solution

先化简一下DDD，用和式表示：

D=(A1  A2  ⋯  An)×(B1,1B1,2⋯B1,nB2,1B2,2⋯B2,n⋮⋮⋱⋮Bn,1Bn,2⋯Bn,n)×(A1A2⋮An)−(C1  C2  ⋯  Cn)×(A1A2⋮An)=(∑i=1nAiBi,1∑i=1nAiBi,2⋯∑i=1nAiBi,n)×(A1A2⋮An)−(C1  C2  ⋯  Cn)×(A1A2⋮An)=∑i=1n∑j=1nAiAjBi,j−∑i=1nAiCi\begin{aligned} D &= \begin {pmatrix} A_1\;A_2\;\cdots\;A_n \end {pmatrix} \times \begin {pmatrix} B_{1,1}&B_{1,2}&\cdots&B_{1,n}\\ B_{2,1}&B_{2,2}&\cdots&B_{2,n}\\ \vdots&\vdots&\ddots&\vdots\\ B_{n,1}&B_{n,2}&\cdots&B_{n,n} \end {pmatrix} \times \begin {pmatrix} A_1\\A_2\\\vdots\\A_n \end {pmatrix} - \begin {pmatrix} C_1\;C_2\;\cdots\;C_n \end {pmatrix} \times \begin {pmatrix} A_1\\A_2\\\vdots\\A_n \end {pmatrix} \\ &= \begin {pmatrix} \sum_{i=1}^{n}A_iB_{i,1}&\sum_{i=1}^{n}A_iB_{i,2}&\cdots&\sum_{i=1}^{n}A_iB_{i,n} \end {pmatrix} \times \begin {pmatrix} A_1\\A_2\\\vdots\\A_n \end {pmatrix} - \begin {pmatrix} C_1\;C_2\;\cdots\;C_n \end {pmatrix} \times \begin {pmatrix} A_1\\A_2\\\vdots\\A_n \end {pmatrix} \\ &= \sum_{i=1}^{n}\sum_{j=1}^{n}A_iA_jB_{i,j}-\sum_{i=1}^{n}A_iC_i \end{aligned} D​=(A1​A2​⋯An​​)×⎝⎜⎜⎜⎜⎛​B1,1​B2,1​⋮Bn,1​​B1,2​B2,2​⋮Bn,2​​⋯⋯⋱⋯​B1,n​B2,n​⋮Bn,n​​⎠⎟⎟⎟⎟⎞​×⎝⎜⎜⎜⎜⎛​A1​A2​⋮An​​⎠⎟⎟⎟⎟⎞​−(C1​C2​⋯Cn​​)×⎝⎜⎜⎜⎜⎛​A1​A2​⋮An​​⎠⎟⎟⎟⎟⎞​=(∑i=1n​Ai​Bi,1​​∑i=1n​Ai​Bi,2​​⋯​∑i=1n​Ai​Bi,n​​)×⎝⎜⎜⎜⎜⎛​A1​A2​⋮An​​⎠⎟⎟⎟⎟⎞​−(C1​C2​⋯Cn​​)×⎝⎜⎜⎜⎜⎛​A1​A2​⋮An​​⎠⎟⎟⎟⎟⎞​=i=1∑n​j=1∑n​Ai​Aj​Bi,j​−i=1∑n​Ai​Ci​​

由于AAA是010101矩阵，可以将其看作有nnn个物品，选择取或不去，第iii给物品取有CiC_iCi​的代价，第iii个和第jjj个同时取会有Bi,jB_{i,j}Bi,j​的贡献。要求使总贡献DDD最大。

对于每个物体建一个点，任意两个物体的组合建一个点。对于组合<i,j><i,j><i,j>，连接其代表的点和第iii个物体的点与第jjj个物体的点。对于物体的点，权值设为−Ci-C_i−Ci​，对于组合的点，权值设为Bi,jB_{i,j}Bi,j​。这样选择一个组合的点，必须将其连向的两个物体的点也选择，即变为最大权闭合子图模型。

建模：

  • 对每个物体iii建一个点pip_ipi​，连接pi→Tp_i\to Tpi​→T流量CiC_iCi​
  • 对每个组合<i,j><i,j><i,j>建一个点pi,jp_{i,j}pi,j​，连接S→pi,jS\to p_{i,j}S→pi,j​流量Bi,jB_{i,j}Bi,j​
  • 对于每个组合<i,j><i,j><i,j>，连边pi,j→pip_{i,j}\to p_ipi,j​→pi​流量∞\infty∞，连边pi,j→pjp_{i,j}\to p_jpi,j​→pj​流量∞\infty∞

跑最大流求最小割，答案为∑i=1n∑j=1nBi,j−最小割\sum_{i=1}^{n}\sum_{j=1}^{n}B_{i,j}-最小割∑i=1n​∑j=1n​Bi,j​−最小割。
