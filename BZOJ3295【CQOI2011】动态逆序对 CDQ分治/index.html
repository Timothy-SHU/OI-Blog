BZOJ3295【CQOI2011】动态逆序对 < CDQ分治 >

Posted on 2018-05-21 Symbols count in article: 807 Reading time ≈ 3 mins.

Problem

【CQOI2011】动态逆序对

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

对于序列AAA，它的逆序对数定义为满足i<ji<ji<j，且Ai>AjA_i>A_jAi​>Aj​的数对(i,j)(i,j)(i,j)的个数。
给111到nnn的一个排列，按照某种顺序依次删除mmm个元素，你的任务是在每次删除一个元素之前统计整个序列的逆序对数。

Input

输入第一行包含两个整数nnn和mmm，即初始元素的个数和删除的元素个数。
以下nnn行每行包含一个111到nnn之间的正整数，即初始排列。
以下mmm行每行一个正整数，依次为每次删除的元素。

Output

输出包含mmm行，依次为删除每个元素之前，逆序对的个数。

Sample Output

HINT

N≤105,  M≤5×104N\le10^5,\;M\le5\times10^4N≤105,M≤5×104

标签：CDQ分治 树状数组 三维偏序

Solution

三维偏序，可上CDQ\mathrm{CDQ}CDQ分治。

首先将删除操作离线，倒着做变成插入，并给每个数编号(x,y,z)(x,y,z)(x,y,z)分别表示其下标、数值、插入时间。需要计算每个元素插入后会多出多少个逆序对，即对于三元组(xi,yi,zi)(x_i,y_i,z_i)(xi​,yi​,zi​)，有多少个j≠ij\ne ij=i满足xj<xi,yj>yi,zj≤zix_j<x_i,y_j>y_i,z_j\le z_ixj​<xi​,yj​>yi​,zj​≤zi​或xj>xi,yj<yi,zj<zix_j>x_i,y_j<y_i,z_j<z_ixj​>xi​,yj​<yi​,zj​<zi​。
两个三维偏序，做两次CDQ\mathrm{CDQ}CDQ分治后统计即可。
