BZOJ1901 Dynamic Rankings <整体二分>

Posted on 2018-05-10 Symbols count in article: 1.3k Reading time ≈ 5 mins.

Problem

Dynamic Rankings

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

给定一个含有nnn个数的序列a[1],a[2],a[3],⋯ ,a[n]a[1],a[2],a[3],\cdots,a[n]a[1],a[2],a[3],⋯,a[n]。
对于给定的i,j,ki,j,ki,j,k，请回答在a[i],a[i+1],a[i+2],⋯ ,a[j]a[i],a[i+1],a[i+2],\cdots,a[j]a[i],a[i+1],a[i+2],⋯,a[j]中第kkk小的数是多少(1≤k≤j−i+1)(1\le k\le j-i+1)(1≤k≤j−i+1)。
在询问中会有操作改变一些a[i]a[i]a[i]的值，改变后，需要针对改变后的aaa继续回答上面的问题。

Input

第一行有两个正整数n,mn,mn,m。
分别表示序列的长度和指令的个数。
第二行有nnn个数，表示a[1],a[2]⋯ ,a[n]a[1],a[2]\cdots,a[n]a[1],a[2]⋯,a[n]，这些数都小于10910^9109。
接下来的mmm行描述每条指令，每行的格式是下面两种格式中的一种。

  • Q  i  j  k  (1≤i≤j≤n,1≤k≤j−i+1)Q\;i\;j\;k\;(1\le i\le j\le n,1\le k\le j-i+1)Qijk(1≤i≤j≤n,1≤k≤j−i+1)表示询问指令，询问a[i],a[i+1],⋯ ,a[j]a[i],a[i+1],\cdots,a[j]a[i],a[i+1],⋯,a[j]中第kkk小的数。
  • C  i  t  (1≤i≤n,0≤t≤109)C\;i\;t\;(1\le i\le n,0\le t\le10^9)Cit(1≤i≤n,0≤t≤109)表示把a[i]a[i]a[i]改变成为ttt。

Output

对于每一次询问，你都需要输出他的答案，每一个输出占单独的一行。

Solution

整体二分经典题，直接上整体二分即可。

考虑将所有询问离线下来，对所有询问同时进行二分答案，二分函数有四个参数l,r,s,tl,r,s,tl,r,s,t，表示答案在[s,t][s,t][s,t]区间内的询问为在询问序列上位置在[l,r][l,r][l,r]之间的所有询问。
如果s=ts=ts=t，则可知询问序列上位置在[l,r][l,r][l,r]间的所有询问答案均为sss。否则，我们需要把[l,r][l,r][l,r]间的所有询问分为前后两部分，前一部分为答案在[s,s+t2][s,\frac{s+t}{2}][s,2s+t​]间的所有询问，后一部分为答案在[s+t2+1,t][\frac{s+t}{2}+1,t][2s+t​+1,t]间的所有询问。这个过程用一棵树状数组判断一下[l,r][l,r][l,r]间每个询问的答案在哪边即可。
然而我们还需要维护修改操作。对于修改操作，我们同样将其加入整体二分。当询问函数将答案限制到[s,t][s,t][s,t]时，只有参数ttt在[s,t][s,t][s,t]间的修改操作才会对这个[l,r][l,r][l,r]间的询问答案产生影响。于是将每个修改拆成两个操作，即在某位置上删除一个数和加入一个数。
用树状数组维护时，若删除或加入的数在[s,s+t2][s,\frac{s+t}{2}][s,2s+t​]间，在树状数组上对应位置−1-1−1或+1+1+1。对于询问，若询问区间为[tl,tr][tl,tr][tl,tr]，那么该询问的numnumnum值为树状数组上[tl,tr][tl,tr][tl,tr]位置上的值之和。这样一来，numnumnum统计的是每个询问区间内小于等于s+t2\frac{s+t}{2}2s+t​的数的个数，若num≥knum\ge knum≥k，则答案一定在[s,s+t2][s,\frac{s+t}{2}][s,2s+t​]间；否则，答案一定大于s+t2\frac{s+t}{2}2s+t​。分治处理即可。

由于整体二分会带来一个log⁡n\log{n}logn，枚举每个询问并用树状数组判断其答案在左边还是右边需要nlog⁡nn\log{n}nlogn，故总复杂度为O(nlog⁡2n)O(n\log^{2}{n})O(nlog2n)。
