BZOJ3594【SCOI2014】方伯伯的玉米田 <树状数组优化DP>

Posted on 2018-03-14 Symbols count in article: 735 Reading time ≈ 3 mins.

Problem

【SCOI2014】方伯伯的玉米田

Time  Limit:  60  Sec\mathrm{Time\;Limit:\;60\;Sec}TimeLimit:60Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

方伯伯在自己的农田边散步，他突然发现田里的一排玉米非常的不美。
这排玉米一共有NNN株，它们的高度参差不齐。
方伯伯认为单调不下降序列很美，所以他决定先把一些玉米拔高，再把破坏美感的玉米拔除掉，使得剩下的玉米的高度构成一个单调不下降序列。
方伯伯可以选择一个区间，把这个区间的玉米全部拔高111单位高度，他可以进行最多KKK次这样的操作。拔玉米则可以随意选择一个集合的玉米拔掉。
问能最多剩多少株玉米，来构成一排美丽的玉米。

Input

第111行包含222个整数N,KN,KN,K，分别表示这排玉米的数目以及最多可进行多少次操作。
第222行包含NNN个整数，第iii个数表示这排玉米，从左到右第iii株玉米的高度aia_iai​。

Output

输出111个整数，最多剩下的玉米数。

Sample Output

Hint

1<N<104,  1<K≤500,  1≤ai≤50001<N<10^4,\;1<K\le500,\;1\le a_i\le50001<N<104,1<K≤500,1≤ai​≤5000

标签：DP 树状数组

Solution

首先，显然每次拔高时，不管从哪里开始拔，区间右边界总是NNN肯定不会使答案变小。有此贪心后，考虑到第iii株时前面拔高jjj次，那么第iii株一定也已拔高jjj次。

令f[i][j]f[i][j]f[i][j]表示考虑前iii株，共拔高了jjj次，最多可以剩下多少。那么有

f[i][j]=max⁡f[p][q]+1        (p<i,  q≤j,  a[p]+q≤a[i]+j)f[i][j]=\max{f[p][q]}+1\;\;\;\;(p<i,\;q\le j,\;a[p]+q\le a[i]+j) f[i][j]=maxf[p][q]+1(p<i,q≤j,a[p]+q≤a[i]+j)

对于DP\mathrm{DP}DP值f[x][y]f[x][y]f[x][y]，将其坐标设为(x,a[x]+y)(x,a[x]+y)(x,a[x]+y)，那么DP\mathrm{DP}DP到f[i][j]f[i][j]f[i][j]时，需要找的是坐标在(0,0)∼(j,a[i]+j−1)(0,0)\sim(j,a[i]+j-1)(0,0)∼(j,a[i]+j−1)范围内的最小值，可以二维树状数组维护。

注意DP\mathrm{DP}DP时第二层循环要倒着循环，以排除后效性，由于二维树状数组的坐标范围不能到000，可以把所有坐标的横纵值强行加111。
