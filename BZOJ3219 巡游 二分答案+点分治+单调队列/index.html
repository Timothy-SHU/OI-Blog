BZOJ3219 巡游 <二分答案+点分治+单调队列>

Posted on 2018-05-02 Symbols count in article: 1.6k Reading time ≈ 6 mins.

Problem

巡游

Time  Limit:  25  Sec\mathrm{Time\;Limit:\;25\;Sec}TimeLimit:25Sec
Memory  Limit:  128  MB\mathrm{Memory\;Limit:\;128\;MB}MemoryLimit:128MB

Description

Tar\mathrm{Tar}Tar国正在准备每年一次的巡游活动。国王将会在一个城市SSS里召集人群，沿着城市间的道路进行游览，最终在一个城市TTT里发表他每年一次的著名演讲。
Tar\mathrm{Tar}Tar国有NNN个城市，由于国家的特殊要求，每两个城市之间存在一条唯一的简单通路。国王希望借着这个机会视察Tar\mathrm{Tar}Tar国的城市建设，因此他提出SSS到TTT的距离不能少于LLL条道路。
同时，国王的私人医生检查了他的身体情况后，断定国王的身体不适合做长途旅行，因此他要求SSS到TTT的距离不能多于RRR条道路。
另外，政府希望跟随国王的人民沿途不仅能看到城市风景，还能看到城市外的美丽乡村。因此每条道路定义了一个魅力值CiC_iCi​，一条路径的魅力值定义为这条路径的中位数。更详细的说法是这样的：将路径上所有边的魅力值排序，得到序列{Ai}\lbrace A_i\rbrace{Ai​}。假设i=2k+c  (0≤c≤1)i=2k+c\;(0\le c\le 1)i=2k+c(0≤c≤1)，中位数就是Ak+1A_{k+1}Ak+1​。
你的任务就是求出魅力值最大的路径，并输出这个魅力值。

Input

第一行是三个整数N,L,RN,L,RN,L,R，表示Tar\mathrm{Tar}Tar国的城市个数、路径的最小和最大长度。
接下来N−1N-1N−1行，每行333个整数Ai,Bi,CiA_i,B_i,C_iAi​,Bi​,Ci​，表示有一条连接AiA_iAi​和BiB_iBi​且魅力值CiC_iCi​的道路。

Output

仅一行，表示最大的魅力值。如果不存在这样的路径，输出−1-1−1。

Solution

稍有码量的点分题。

首先策略是二分答案+点分治验证二分答案+点分治验证二分答案+点分治验证，二分答案魅力值，将所有的边权变为111和−1-1−1，分别表示大于等于魅力值和小于魅力值。这样验证问题转化为判断是否有一条长度在[L,R][L,R][L,R]间路径使得边权和大于等于000。

这个判断过程可以用点分治实现。对于每个分治中心，只考虑经过其的路径。在其点分树的不同子树中找两个点，使得其到分治中心的路径长度和在[L,R][L,R][L,R]之间，可以用两个桶，分别存已枚举的子树和当前子树中各个深度的最大路径边权和，需要用单调队列维护一下。这部分有些细节需要注意。

此题卡常，注意一些减小常数的细节：

  • 二分答案时，将原边权记下来排序，在排好的数组上二分，这样只会二分到边权值
  • 一开始将点分树记下来，记录所有分治中心，这样每次二分checkcheckcheck可以不用重新找重心
  • 点分时在当前分治中心统计答案时用BFS\mathrm{BFS}BFS
  • 预处理点分时在当前分治中心BFS\mathrm{BFS}BFS，若下一个点的距离大于RRR则退出
