BZOJ1430 小猴打架 < Prufer序列+组合数学 >

Posted on 2018-03-24 Symbols count in article: 397 Reading time ≈ 1 mins.

Problem

小猴打架

Time  Limit:  5  Sec\mathrm{Time\;Limit:\;5\;Sec}TimeLimit:5Sec
Memory  Limit:  162  MB\mathrm{Memory\;Limit:\;162\;MB}MemoryLimit:162MB

Description

一开始森林里面有NNN只互不相识的小猴子，它们经常打架，但打架的双方都必须不是好朋友。每次打完架后，打架的双方以及它们的好朋友就会互相认识，成为好朋友。经过N−1N-1N−1次打架之后，整个森林的小猴都会成为好朋友。 现在的问题是，总共有多少种不同的打架过程。 比如当N=3N=3N=3时，就有{1−2,1−3}\lbrace1-2,1-3\rbrace{1−2,1−3} {1−2,2−3}\lbrace1-2,2-3\rbrace{1−2,2−3} {1−3,1−2}\lbrace1-3,1-2\rbrace{1−3,1−2} {1−3,2−3}\lbrace1-3,2-3\rbrace{1−3,2−3} {2−3,1−2}\lbrace2-3,1-2\rbrace{2−3,1−2} {2−3,1−3}\lbrace2-3,1-3\rbrace{2−3,1−3}六种不同的打架过程。

Input

一个整数NNN，N≤106N\le10^6N≤106

Output

一行，方案数mod  9999991\mod{9999991}mod9999991。

Sample Output

Solution

由Prufer序列\mathrm{Prufer}序列Prufer序列可知，一棵有nnn个点的树与一个长为n−2n-2n−2的序列一一对应。那么确定出这样的序列的方案数即可确定最后形成的树的形态。易知这样的序列共有nn−2n^{n-2}nn−2个。再考虑构建树的顺序，即为n−1n-1n−1个元素的全排列数，总数为(n−1)!(n-1)!(n−1)!。故答案为 nn−2−(n−1)!mod  9999991n^{n-2}-(n-1)!\mod{9999991}nn−2−(n−1)!mod9999991。
