【BJWC2017】神秘物质

Time  Limit:  20  Sec\mathrm{Time\;Limit:\;20\;Sec}TimeLimit:20Sec
Memory  Limit:  256  MB\mathrm{Memory\;Limit:\;256\;MB}MemoryLimit:256MB

Description

21ZZ\mathrm{21ZZ}21ZZ年，冬。
小诚退休以后，不知为何重新燃起了对物理学的兴趣。他从研究所借了些实验仪器，整天研究各种微观粒子。
这一天，小诚刚从研究所得到了一块奇异的陨石样本，便迫不及待地开始观测。在精密仪器的视野下，构成陨石的每个原子都无比清晰。
小诚发现，这些原子排成若干列，每一列的结构具有高度相似性。于是，他决定对单独一列原子进行测量和测试。
被选中的这列共有NNN个顺序排列的原子。最初，第iii个原子具有能量EiE_iEi​。随着时间推移和人为测试，这列原子在观测上会产生两种变化：

  • merge  x  emerge\;x\;emergexe：当前第xxx个原子和第x+1x+1x+1个原子合并，得到能量为eee的新原子；
  • insert  x  einsert\;x\;einsertxe：在当前第xxx个原子和第x+1x+1x+1个原子之间插入一个能量为eee的新原子。

对于一列原子，小诚关心的是相邻一段中能量最大和能量最小的两个原子的能量差值，称为区间极差。因此，除了观测变化外，小诚还要经常统计这列原子的两类数据：

  • max  x  ymax\;x\;ymaxxy：当前第xxx到第yyy个原子之间的任意子区间中区间极差的最大值；
  • min  x  ymin\;x\;yminxy：当前第xxx到第yyy个原子之间的任意子区间中区间极差的最小值。

其中，子区间指的是长度至少是222的子区间。
小诚坚信这项研究可以获得诺贝尔物理学奖。为了让小诚早日了结心愿，你能否帮助他实现上述的观测和测量呢？

Input

第一行，两个整数N,MN,MN,M，分别表示最初的原子数目和事件总数。
第二行，NNN个整数E1,E2,⋯ ,ENE_1,E_2,\cdots,E_NE1​,E2​,⋯,EN​，由空格隔开，依次表示每个原子的能量。
接下来MMM行， 每行为一个字符串和两个整数， 描述一次事件，格式见题目描述。

Output

输出若干行， 按顺序依次表示每次maxmaxmax和minminmin类事件的测量结果。

标签：Splay

Solution

Splay\mathrm{Splay}Splay基础题。

对于insertinsertinsert操作，直接插入即可。
对于mergemergemerge操作，先删除第xxx和x+1x+1x+1个数，然后在第x−1x-1x−1个数之后插入eee。
对于maxmaxmax询问，易知其答案为该区间的最大数减最小数，于是在Splay\mathrm{Splay}Splay上维护区间最大值和最小值即可。
对于minminmin询问，其答案一定存在于相邻两数的差之中，在Splay\mathrm{Splay}Splay上维护区间相邻两数差的最小值。具体地，在第iii个数的位置维护第iii个数和第i+1i+1i+1个数的差的绝对值，这样询问[l,r][l,r][l,r]变为询问[l,r−1][l,r-1][l,r−1]区间中差的最小值。注意在insertinsertinsert和mergemergemerge时会导致插入位置相邻的数的差值发生变化，需要同时维护。
