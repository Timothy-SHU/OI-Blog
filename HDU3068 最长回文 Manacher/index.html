HDU3068 最长回文 < Manacher >

Posted on 2017-09-25 Symbols count in article: 452 Reading time ≈ 2 mins.

Problem

最长回文

Time Limit: 4000/2000MS(Java/Others)4000/2000 MS (Java/Others)4000/2000MS(Java/Others)
Memory Limit: 32768/32768K(Java/Others)32768/32768 K (Java/Others)32768/32768K(Java/Others)

Problem Description

给出一个只由小写英文字符a,b,c⋯y,za,b,c\cdots y,za,b,c⋯y,z组成的字符串SSS, 求SSS中最长回文串的长度.
回文就是正反读都是一样的字符串, 如abaabaaba, abbaabbaabba等

Input

输入有多组casecasecase,不超过120120120组,每组输入为一行小写英文字符a,b,c⋯y,za,b,c\cdots y,za,b,c⋯y,z组成的字符串SSS
两组casecasecase之间由空行隔开(该空行不用处理)
字符串长度len≤110000len\le 110000len≤110000

Output

每一行一个整数xxx,对应一组casecasecase,表示该组casecasecase的字符串中所包含的最长回文长度.

Sample Input

Sample Output

标签：Manacher

Solution

ManacherManacherManacher板子题。
不懂ManacherManacherManacher可以戳此博客。
简单来说，ManacherManacherManacher就是记录以每个位置为中心的回文串的半径长度，这样在后面计算的时候可以根据对称性，利用前面的结果加速，找更长匹配时暴力扩展，复杂度O(n)O(n)O(n)。由于此算法仅能对付长度为奇数的回文串（毕竟你算的是以每个点为中心的回文串），故先在每两个字符间插入一个占位符。
