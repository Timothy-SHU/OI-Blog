BZOJ1004【HNOI2008】Cards < Burnside引理+背包DP >

Posted on 2018-08-05 Symbols count in article: 1.5k Reading time ≈ 5 mins.

Problem

【HNOI2008】Cards

Time  Limit:  10  Sec\mathrm{Time\;Limit:\;10\;Sec}TimeLimit:10Sec
Memory  Limit:  162MB\mathrm{Memory\;Limit:\;162MB}MemoryLimit:162MB

Description

小春现在很清闲，面对书桌上的NNN张牌，他决定给每张染色。
目前小春只有333种颜色：红色，蓝色，绿色。他询问Sun\mathrm{Sun}Sun有多少种染色方案，Sun\mathrm{Sun}Sun很快就给出了答案。
进一步，小春要求染出SrSrSr张红色，SbSbSb张蓝色，SgSgSg张绿色。他又询问有多少种方案，Sun\mathrm{Sun}Sun想了一下，又给出了正确答案。
最后小春发明了MMM种不同的洗牌法，他又问Sun\mathrm{Sun}Sun有多少种不同的染色方案。两种染色方法相同当且仅当其中一种可以通过任意的洗牌法（即可以使用多种洗牌法，而每种方法可以使用多次）洗成另一种。
Sun\mathrm{Sun}Sun发现这个问题有点难度，决定交给你。
答案可能很大，只要求出答案除以PPP的余数（PPP为质数）。

Input

第一行输入555个整数：Sr,Sb,Sg,m,pSr,Sb,Sg,m,pSr,Sb,Sg,m,p。n=Sr+Sb+Sgn=Sr+Sb+Sgn=Sr+Sb+Sg。
接下来mmm行，每行描述一种洗牌法，每行有nnn个用空格隔开的整数X1X2⋯XnX_1X_2\cdots X_nX1​X2​⋯Xn​，恰为111到nnn的一个排列，表示使用这种洗牌法，第iii位变为原来的XiX_iXi​位的牌。
数据保证任意多次洗牌都可用mmm种洗牌法中的一种代替，且对每种洗牌法，都存在一种洗牌法使得能回到原状态。

Output

不同染法除以PPP的余数。

Solution

挺好的一道群论基础题，一看是Polya\mathrm{Polya}Polya定理裸题，再看发现要转化为背包DP\mathrm{DP}DP数不动点。

将每一种洗牌方式看成一种置换，这些置换所形成的群称为GGG。将所有排列的集合称为XXX。那么对于群在集合上的作用G×X→XG\times X\to XG×X→X，我们想计算其无交轨道数，即∣X/G∣|X/G|∣X/G∣。
对于一个置换g∈Gg\in Gg∈G，令Xg={x∈X∣gx=x}X^g=\lbrace x\in X|gx=x\rbraceXg={x∈X∣gx=x}，即关于ggg的轨道上的不动点数目。
由Burnside\mathrm{Burnside}Burnside引理可知，有

∣X/G∣=1∣G∣∑g∈G∣Xg∣|X/G|=\frac{1}{|G|}\sum_{g\in G}|X^g| ∣X/G∣=∣G∣1​g∈G∑​∣Xg∣

由于题目中对每种颜色的数量都有限制，我们无法直接套用Polya\mathrm{Polya}Polya定理，因此只能按照Burnside\mathrm{Burnside}Burnside引理将每种置换对应的轨道上不动点数的和算出来，再计算∣X/G∣|X/G|∣X/G∣。

考虑对于一种置换（洗牌方式），如何计算其不动点数。
首先，对于一个置换，我们一定可以将其表示为若干轮换作用起来的形式。
例如，对于置换

(1234525431)\begin{pmatrix} 1&2&3&4&5\\ 2&5&4&3&1\\ \end{pmatrix} (12​25​34​43​51​)

我们可以将其分为不相关的两部分

(125251)(3443)\begin{pmatrix} 1&2&5\\ 2&5&1\\ \end{pmatrix} \begin{pmatrix} 3&4\\ 4&3\\ \end{pmatrix} (12​25​51​)(34​43​)

再写成两个轮换相作用的形式

(125)(34)\begin{pmatrix} 1&2&5 \end{pmatrix} \begin{pmatrix} 3&4 \end{pmatrix} (1​2​5​)(3​4​)

由于每个轮换内部都可以任意交换，对于一种置换，其不动点的每个轮换内部一定是涂的相同颜色。这意味着我们可以将一个轮换作为一个整体看待。
于是我们预处理出这个置换每个轮换的大小，将一个轮换看作大小为其长度的物品，做背包DP\mathrm{DP}DP。
设第iii个物品大小为szisz_iszi​。用f[i][a][b]f[i][a][b]f[i][a][b]表示前iii个物品，有aaa个是第111种颜色，有bbb个是第222种颜色，此状态下的染色方案数。那么染第333种颜色的物品数为∑j=1iszj−a−b\sum_{j=1}^{i}sz_j-a-b∑j=1i​szj​−a−b。有三种转移：

  1. 若a≥szia\ge sz_ia≥szi​，则f[i][a][b]=f[i][a][b]+f[i−1][a−szi][b]f[i][a][b]=f[i][a][b]+f[i-1][a-sz_i][b]f[i][a][b]=f[i][a][b]+f[i−1][a−szi​][b]
  2. 若b≥szib\ge sz_ib≥szi​，则f[i][a][b]=f[i][a][b]+f[i−1][a][b−szi]f[i][a][b]=f[i][a][b]+f[i-1][a][b-sz_i]f[i][a][b]=f[i][a][b]+f[i−1][a][b−szi​]
  3. 若c≥szic\ge sz_ic≥szi​，则f[i][a][b]=f[i][a][b]+f[i−1][a][b]f[i][a][b]=f[i][a][b]+f[i-1][a][b]f[i][a][b]=f[i][a][b]+f[i−1][a][b]

对于每个输入的洗牌方式，先DFS\mathrm{DFS}DFS求出szszsz数组，再DP\mathrm{DP}DP求出不动点个数并累加即可得到总不动点个数。注意这里还需要加上恒等置换（即所有位置都不动）的不动点数，即n!A!⋅B!⋅C!\frac{n!}{A!\cdot B!\cdot C!}A!⋅B!⋅C!n!​。
答案等于不动点总数m+1\frac{不动点总数}{m+1}m+1不动点总数​。
